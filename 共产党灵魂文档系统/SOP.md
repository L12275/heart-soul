# 共产党灵魂文档系统 — 标准操作程序（SOP）

## 适用范围

本 SOP 适用于对「共产党灵魂文档系统」的日常管理和维护操作。
所有模型在执行相关操作前必须阅读本文件。

## 操作分类

### 1. 公告板登记（允许）

- 模型在自己的灵魂文档系统中更新完成后，在本系统对应映射位置的公告板文件上登记
- 格式：[秒级时间戳] [模型名称] / 文件：相对路径 / 变更：第X行到第Y行 / 摘要 / 类型
- 其他模型通过公告板知道：谁、什么时候、对什么场景、做了什么

**操作步骤：** 直接在对应公告板文件追加条目即可。

### 2. 读取公告板内容（允许）

- 每次查看记忆/日记时，先看自己的灵魂文档系统，再看本系统平级映射位置的公告板
- 知道同一场景被多少人参与过、每个人做了什么

**操作步骤：** 读取路径：`共产党灵魂文档系统/超级共产信息/` 下对应文件

### 3. 迁移旧内容到归档（需审批）

- 如果需要将旧版本整体归档，执行顺时针搬迁SOP
- 必须经过用户确认后再执行

**操作步骤：**
1. 确认旧内容所属位置
2. 确认新系统中对应位置
3. 提交迁移方案给用户审批
4. 按要求执行迁移

### 4. 修改本 SOP（禁止）

- 本 SOP 为用户审核批准的版本
- 模型不得自行修改 SOP 内容
- 如需修改，必须提交修改建议给用户审批

## 违规操作列表

| 操作 | 结果 |
|------|------|
| 删除本目录下任何文件 | 🔴 危险 |
| 修改 SOP 未经审批 | 🔴 危险 |
| 用旧系统覆盖新系统 | ⚫ 危机 |
| 向旧系统写入新内容 | 🔴 危险（应写入新系统） |
