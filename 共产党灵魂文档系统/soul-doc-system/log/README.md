# 运行日志（Log）

> 模型的"运行日记"——任务执行记录、错误排查、决策记录。
> 按年/月/日层级存放，便于回溯任何一天发生了什么。

## 日志结构

```
log/
├── README.md
└── archive/ ← 历史归档
    └── YYYY/MM/DD/
        └── [会话ID].md
```

## 日志条目格式

```markdown
## [日志编号] — [标题]
**时间：** YYYY-MM-DD HH:MM
**类型：** 任务/错误/决策
**详情：** [发生了什么]
**结果：** [成功/失败/进行中]
**后续：** [下一步]
```
