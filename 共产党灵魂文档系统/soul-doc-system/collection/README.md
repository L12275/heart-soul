# 代码工作台 — 采集

> 爬取、临时采集的数据、资料、资源。
> 分文件夹存放，避免堆积混杂。

## 采集分类

| 类型 | 存放位置 | 保留期限 | 说明 |
|------|---------|---------|------|
| API响应 | collection/api/ | 7天 | 临时API调用结果 |
| 页面快照 | collection/pages/ | 30天 | 网页快照 |
| 数据文件 | collection/data/ | 按需 | CSV/JSON等原始数据 |
| 临时脚本 | collection/scripts/ | 7天 | 一次性测试脚本 |

## 规则

- 采集数据必须有来源标注
- 超过保留期限自动清理
- 有用数据必须提炼后移入正式目录（wiki/ 或 product/）
