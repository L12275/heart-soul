# MCP 工具

> **自主LLM代理的主动工具发现。**
> MCP（Model Context Protocol）让模型能自主发现和使用外部工具。

## 核心项目

**MCP-Zero**: https://github.com/xfey/MCP-Zero
- 自主LLM代理的主动工具发现
- 在需要时可自行用MCP-Zero发现和集成新工具
- 无需预先配置，模型自主探索可用工具

## 使用方式

1. 模型在需要新工具时，通过 MCP-Zero 主动发现
2. 自动集成到工作流中
3. 不需要时自动释放
