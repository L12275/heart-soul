GA 原子工具集 — 9 个独立工具

> 从 GenericAgent 学习的 9 个原子工具，每个都是独立脚本。
> 基于 ljqCtrl.py（Windows）和 macljqCtrl.py（macOS）实现。
> 参考：https://github.com/lsdefine/GenericAgent

---

## 工具清单

| # | 工具 | 脚本文件 | GA 底层函数 | 说明 |
|---|------|---------|------------|------|
| 1 | 截图 | `tools/screenshot.py` | GrabWindow, GrabWindowBg, ScreenCapAt | 全屏/窗口/区域/点周围截图 |
| 2 | 鼠标点击 | `tools/click.py` | Click(check=True) | 硬件级点击+像素验证 |
| 3 | 鼠标移动 | `tools/mouse_move.py` | SetCursorPos | 移动到物理坐标 |
| 4 | 鼠标滚动 | `tools/mouse_scroll.py` | mouse_event(MOUSEEVENTF_WHEEL) | 正=向上，负=向下 |
| 5 | 键盘输入 | `tools/keyboard_input.py` | Press + pyperclip | 文字粘贴或组合键 |
| 6 | 窗口激活 | `tools/window_activate.py` | Activate(hwnd) | 假Alt-up绕过前台锁 |
| 7 | 拖放 | `tools/drag_drop.py` | SetCursorPos + MouseDown/Up | 从源到目标拖动 |
| 8 | 窗口信息 | `tools/window_info.py` | FindWindow, GetWindowRect | 枚举窗口/获取信息 |
| 9 | 快捷键 | `tools/hotkey.py` | Press(cmd) | 组合键如 ctrl+v |

---

## 用法示例

```bash
# 1. 截图
python tools/screenshot.py --window "Chrome" -o chrome.png

# 2. 点击（带验证）
python tools/click.py 100 200

# 3. 移动鼠标
python tools/mouse_move.py 500 300

# 4. 滚动（向下3格）
python tools/mouse_scroll.py 500 300 -3

# 5. 输入文字
python tools/keyboard_input.py "hello world" --click 200 150

# 6. 激活窗口
python tools/window_activate.py "Chrome"

# 7. 拖放
python tools/drag_drop.py 100 200 300 400

# 8. 窗口信息
python tools/window_info.py --all

# 9. 快捷键
python tools/hotkey.py ctrl+v
python tools/hotkey.py alt+tab
```

## 四条铁律

| 铁律 | 说明 |
|------|------|
| **物理坐标** | 所有坐标必须是物理像素 = 截图像素坐标 |
| **禁 pyautogui** | pyautogui 污染 win32api |
| **先激活** | 操作前必须 Activate 窗口 |
| **像素验证** | Click 后检查变化，0%=点歪了 |

## 底层实现

所有工具基于 `ljqCtrl.py`（Windows）或 `macljqCtrl.py`（macOS），
通过 `import ljqCtrl` 调用硬件级 Win32API：
- `win32api.mouse_event` — 硬件级鼠标事件（非 PostMessage）
- `win32api.keybd_event` — 硬件级键盘事件
- `win32gui.SetForegroundWindow` + 假 Alt-up — 绕过前台锁
- `SetProcessDPIAware` + dpi_scale — DPI 感知

详细实现见 `ljqCtrl.py` 和 `Win32后端参考.md`。
