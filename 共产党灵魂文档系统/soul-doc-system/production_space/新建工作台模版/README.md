# 新建工作台模版

> 本模版用于创建新的专用工作台。
> 使用流程：发现需要新工作台 → 复制本模版 → 重命名 → 填充内容。

---

## 使用步骤

1. 复制整个 `新建工作台模版/` 文件夹
2. 重命名为新工作台名称（如 `写作工作台`）
3. 修改 `README.md` 中的工作台描述
4. 修改 `diary/README.md` 中的 L0 铁律和场景索引
5. 修改 `memory/context/L0.md` 中的核心规则
6. 根据需要调整 `desktop/`、`workflow/` 等子文件夹

## 工作台结构说明

| 文件夹 | 用途 |
|--------|------|
| desktop/ | 工作台式桌面（便签、工作流、联系簿、收藏夹） |
| diary/ | 时间轴日记（实时纪实） |
| memory/ | 5层记忆（L0-L4 + daily 快照） |
| sop/ | 可复用 SOP |
| workflow/ | 标准工作流 |
| product/ | 最终产物（分类存放） |
| temp/ | 临时文件（下载/临时获取，需记录用途） |
