通用智能体

> **通用任务执行智能体模板。已通过基础验证。**

## 用途

适用于大多数通用任务场景，提供标准的智能体执行框架。

## 适用场景

- 日常任务执行
- 信息收集和处理
- 简单决策支持

## 不适用场景

- 需要高度专业领域知识的任务
- 需要多智能体协作的复杂任务
- 需要深度定制配置的场景

## 配置说明

见 `通用智能体配置.json`

## 使用方法

1. 从衣柜复制本模板到 `大厅/活跃/[你的智能体名称]/`
2. 在副本中微调参数
3. 启动运行（穿上衣服）

## 已知限制

- 当前为v1.0，基础功能完整但未经过复杂场景验证
- 子代理能力有限，复杂任务需要手动拆分
