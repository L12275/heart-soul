# 过河桥 — 文件抬头同步脚本

> **此过河桥是 文件抬头同步脚本 的基因片段。**
> 此脚本用于当文件抬头模板更新后，批量更新所有文件的抬头。

---

## 1. 用户原话重述

用户要求：
- 当文件抬头模板更新后，批量更新所有文件
- 脚本读取模板，遍历所有.md文件，将旧抬头替换为新抬头
- [CENTRAL_PATH] 由模型在加载时动态替换，脚本只处理静态抬头内容

---

## 2. 理解要点

**同步脚本的核心职责**：
1. **读取模板** — 从文件抬头模板.md读取最新抬头
2. **遍历文件** — 遍历所有.md文件
3. **替换抬头** — 用正则表达式匹配旧抬头，替换为新抬头
4. **保留内容** — 只替换抬头，不改变文件内容

**正则表达式**：
- 匹配模式：（DOTALL模式，匹配多行）
- 替换：新抬头 + 原文件内容

---

## 3. 可能的歧义

| 歧义 | 说明 | 解决 |
|------|------|------|
| 脚本是否替换[CENTRAL_PATH] | 脚本只处理静态抬头 | [CENTRAL_PATH] 由模型在加载时动态替换 |
| 脚本是否覆盖自定义抬头 | 所有文件都用统一抬头 | 是的，统一抬头是系统规范 |
| 脚本运行频率 | 什么时候运行？ | 模板更新后立即运行 |

---

## 4. 我的判断

**使用同步脚本的标准动作**：
1. 文件抬头模板更新后 → 运行此脚本
2. 脚本自动遍历所有.md文件
3. 替换旧抬头为新抬头
4. 确认替换成功

**重建同步脚本的标准动作**：
1. 从文件抬头模板.md复制抬头内容
2. 编写遍历+替换逻辑
3. 测试脚本
4. 将本过河桥放入
