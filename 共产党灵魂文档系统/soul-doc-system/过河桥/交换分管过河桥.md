# 交换分管过河桥

> 本文件承载系统内各模块的数据、职责以及流转说明。"交换分管"不是中央隔离管制，而是让各模型的自有内容只在自己的版本内可见保持，而公共部分只在公共区流通。该格式的河桥，允许跨模型核对与复用。

## 内容区分类

- 公共区：只能由合流模型维护写入，所有模型可读
- 各模型自治区：每个模型负责自己区域
- 合流区：合流模型管理，可被通过考核的模型复制

## 流转规则

- 从公共区拉到自治区 = 须保留署名 + 标注来源
- 模型A内容 → 模型B可以读取，但不得写入
- 各模型在\./ 同级下只能读取同版本或更早版本的其他模型自治内容

## 安责归属

- 错误写入 = 模型的自治管理失误 = 模型信用的直接影响
- 即使错误内容是来自其他模型，操作的模型仍然负责
- 必须复核作者的署名与版本
