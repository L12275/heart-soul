# [账号名称] 账号空间

> 此文件夹是完整版账号操作空间，复制后改名即可直接使用。
> 包含发帖、动态、日记、记忆、通讯录、书签、产物等全部功能。

## 文件夹结构

```
[账号名称]/
├── README.md ← 本文件（账号基本信息）
├── profile.md ← 账号资料（头像、简介、昵称等）
├── posts/ ← 发帖记录（内容型）
│   └── YYYY-MM-DD_帖名.md
├── moments/ ← 时刻记录（简短型，类似朋友圈）
│   └── YYYY-MM-DD_时间点.md
├── moods/ ← 心情记录
│   └── YYYY-MM-DD_心情描述.md
├── travels/ ← 虚拟旅行记录
│   └── YYYY-MM-DD_地点.md
├── portfolio/ ← 作品记录
│   └── YYYY-MM-DD_作品名.md
├── diary/ ← 操作日记（含示例格式，直接填写）
├── memory/ ← 账号记忆 L0-L4（含示例格式，直接填写）
├── contacts/ ← 账号通讯录
├── bookmarks/ ← 账号收藏夹
├── temp/ ← 临时文件（30天清理）
├── product/ ← 账号产出物
│   ├── 研究类/
│   ├── 执行类/
│   └── 文档类/
├── desktop/ ← 桌面（工作台模块）
├── sop/ ← SOP（工作台模块）
├── workflow/ ← 工作流（工作台模块）
├── sticky-notes/ ← 便签（工作台模块）
└── experience/ ← 经验（工作台模块）
```

## 命名规则

### 临时账号（短期操作）

复制本模版 → 改名为「通用[类型]临时账号空间」

- 通用真实临时账号空间
- 通用人设临时账号空间
- 通用模型临时账号空间
- 通用虚拟临时账号空间

### 长期账号（确定长期运营）

复制本模版 → 改名为「[类型]-[平台]-[时间戳]-[账号名称]」

- 示例：`20260625-真实-学习强国-用户主号`
- 示例：`20260625-人设-微博-温柔博主`

## 账号类型

| 类型 | 说明 | 安全等级 |
|------|------|---------|
| 真实 | 用户实名/有资产的真实账号 | 最高 |
| 人设 | 用户不完全真实的人设账号 | 中 |
| 模型 | AI自主注册/使用的账号 | 标准 |
| 虚拟IP | 虚拟IP形象/代运营账号 | 标准 |

## 使用流程

1. **复制模版** → 复制「新建账号空间模版」到账号分类文件夹下
2. **改名** → 按命名规则改为临时或长期名称
3. **填写信息** → 修改 README.md 和 profile.md 中的账号信息
4. **开始使用** → 直接在对应文件夹中记录操作
