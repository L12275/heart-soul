## L0 层 硬编码核心

> L0 = 账号的基本规则和行为准则，不可修改。

### 什么是L0
L0是该账号的最底层规则，所有操作都不能违反L0。

### L0写什么
1. **账号核心定位** — 这个账号是做什么的
2. **安全红线** — 绝对不能做的事
3. **操作禁令** — 不允许的操作类型
4. **账号基本类型** — 真实/人设/虚拟/虚拟IP
