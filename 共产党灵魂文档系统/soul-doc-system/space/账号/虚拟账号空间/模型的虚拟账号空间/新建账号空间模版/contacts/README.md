# 通讯录

> 通讯录 = 模型的"人脉网络"，用于快速通讯、组网、协作、提效。

## 通讯录条目格式

| 字段 | 说明 |
|------|------|------|---------|
| 姓名/组织 | 联系人名称 |
| 角色 | 什么角色 |
| 联系方式 | 邮箱/电话/网站/社交 |
| 工作场景 | 哪个工作台相关 |

## 通讯记录格式

| 字段 | 说明 |
|------|-----|------|
| 通讯编号 | 唯一标识 |
| 时间 | YYYY-MM-DD HH:MM |
| 类型 | 日常/会议/协作/咨询 |
| 通讯方式 | 邮件/电话/即时通讯/会议 |
| 内容摘要 | 沟通了什么 |

## 规则
1. **模型自己查到的联系方式要记录** — 不记录 = 下次还要问用户
2. **用户提供的联系方式要记录** — 不记录 = 严重浪费
3. **立即记录** — 通讯结束后马上写入通讯记录
4. **定期整理** — 清理失效联系方式，更新变更信息
