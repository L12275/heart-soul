# 虚拟账号空间

> AI自己的账号或代运营的虚拟IP账号记录区。
> 包含两种账号：模型的虚拟账号 + 虚拟IP形象账号。

## 文件夹结构

```
虚拟账号空间/
├── README.md                    ← 本文件
├── 模型的虚拟账号空间/           ← AI自主注册/使用的账号
│   └── [平台-AI账号名-YYYYMMDD]/
│       └── 新建账号空间模版/    ← 复制此模版
└── 虚拟IP形象账号空间/           ← 虚拟IP形象/代运营账号
    └── [IP名称-平台-YYYYMMDD]/
        └── 新建账号空间模版/    ← 复制此模版
```

## 模型的虚拟账号空间

- AI自己在网上探索、注册、使用的账号
- 模型自主决定是否注册、如何使用
- 按平台规则操作，不违反平台条款

## 虚拟IP形象账号空间

- 用户让AI长期建设的虚拟IP形象
- 或用户已有的虚拟IP形象/账号交给AI代运营
- 需要长期记录的，复制「新建工作台模版」到「工作台/」

## 初始化流程

1. 确定账号类型（模型虚拟/虚拟IP）
2. 复制对应分类的「新建账号空间模版」
3. 改名为「平台-名称-时间戳」
4. 如果是长期运营 → 额外复制「新建工作台模版」到「工作台/」
5. 开始记录
