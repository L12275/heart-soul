# 通用-待细分

> 平台临时记录区。
> 不确定平台是否长期操作时，默认在此记录。
> 确认长期操作后，迁移到独立平台文件夹。

## 用途

- 第一次接触某平台，不确定是否长期使用
- 偶尔浏览，不打算持续操作
- 观察性记录，不需要深度工作台

## 升级条件

当确认某个平台需要长期操作时：

1. 复制 `production_space/新建工作台模版/` 到 `space/平台/`
2. 改名为 `[类型]-[平台]-[YYYYMMDD]`
3. 把本文件夹中该平台的 memory/ 和 diary/ 迁移过去
4. 迁移后清除本文件夹中该平台的内容

## 命名规范

- 学习平台：`学习平台-[平台名]-[YYYYMMDD]`
- 社交平台：`社交平台-[平台名]-[YYYYMMDD]`
- 代码平台：`代码平台-[平台名]-[YYYYMMDD]`
- 视频平台：`视频平台-[平台名]-[YYYYMMDD]`
- 评论区：`评论区-[平台名]-[YYYYMMDD]`

## 记录方式

直接在 `通用-待细分/diary/` 和 `通用-待细分/memory/` 下按日期记录即可。
不需要创建独立平台文件夹。
