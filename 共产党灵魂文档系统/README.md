# 共产党灵魂文档系统

> 中央灵魂文档系统的**公共信息交换机/协作中台**。
> 六环架构中的第二环（共产中心）。
>
> **（旧）标记的使用时机**：仅在系统需要更新迁移时临时使用。
> 迁移完成后立即去掉（旧）标记，恢复为本文件夹的正常状态。

## 五环定位

| 环 | 名称 | 说明 |
|----|------|------|
| ① | 用户中心 | 用户的原话方案驱动 |
| ② | 共产中心 | **本系统 = 共产党灵魂文档系统**，公共信息交换机 |
| ③ | 主灵魂中心 | 各模型自建的主灵魂文档系统 |
| ④ | 生产空间中心工作台 | 产出物、SOP、工作流、深度研究 |
| ⑤ | 空间中心 | 平台空间 + 账号空间 |
| ⑥ | 子角色中心 | 各模型自建的子角色灵魂系统 + 团队协作 |

## 核心功能

1. **超级共产信息公告板** — 各模型自愿登记可公知内容的变更
2. **公共查阅区** — 各模型可查阅同场景下其他模型的工作记录
3. **合流机制** — 智慧模型定期合流公告板为正式公共查阅版
4. **团队协作** — AI子代理协作专区
