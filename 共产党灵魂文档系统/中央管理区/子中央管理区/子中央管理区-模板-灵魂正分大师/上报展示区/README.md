上报展示区

> 子中央管理区向主中央管理区汇报的展示区。
> 主智能体先看这里快速了解概况，然后必须完整查看全部内容进行验证。

## 为什么需要上报展示区

- 子中央管理区需要向主中央汇报自己的状态
- 但汇报文件 ≠ 完整内容
- 主智能体先看上报展示区快速了解，然后必须完整查看全部内容

## 文件格式

```
上报展示区/
└── YYYYMMDD-HHMM-汇报名.md
```

## 汇报内容模板

```markdown
# 子中央汇报 — [子中央管理区名称]

> 汇报时间：YYYYMMDD-HHMM

## 当前状态

- 子角色：_
- 当前任务：_
- 运行状态：_

## 待办摘要

_（列出当前待办）_

## 进度摘要

_（列出当前进度）_

## 需要主中央关注的事项

_（列出需要主中央处理的问题）_
```

## 与主中央管理区的关系

- 子中央管理区的工作产出同步到这里
- 主中央智能体先看这里，然后完整查看子中央管理区全部内容
- 主中央智能体发现问题直接在子中央管理区中修改
