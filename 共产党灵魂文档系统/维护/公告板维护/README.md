公告板维护

> 锦衣卫的公告板维护职责——审查、合流、清理BBS公告板。

## 维护内容

1. **审帖** — 审查各模型在公告板上的新登记
2. **合流** — 将已审查的公告合流为正式公共查阅版
3. **清理** — 清理过期/无效/格式错误的登记
4. **索引更新** — 更新公告板索引

## 审帖标准

- 时间戳格式正确（YYYY-MM-DDTHH:MM）
- 文件路径可识别
- 变更描述准确
- 类型选择正确
- 路径映射正确

不合规 → 标记为"待修正"，通知对应模型

## 合流规则

- 同一场景的多条公告 → 合并为一个合流文件
- 按时间排序
- 保留原始公告的链接引用
- 合流版放在 `超级共产信息/合流版/` 目录

## 清理规则

- 超过30天的已合流公告 → 清理
- 格式错误的登记 → 清理
- 重复登记 → 清理
- 清理后更新公告板索引

## 维护日志

每次维护记录到 `维护/维护日志/YYYY-MM-DD.md`
