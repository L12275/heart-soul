# 通用生产空间

> 工作室、工厂、公司、作坊、创意中心 — 结构相同，仅用途标签不同。
> 除非需要严格区分不同类型的工作流，否则统一使用"工作室"即可。

## 类型说明

| 类型 | 适用场景 |
|------|---------|
| 工作室 | 单一深度领域专注（默认推荐） |
| 工厂 | 大规模标准化产出（多任务并行） |
| 公司 | 综合项目管理（多部门协作） |
| 作坊 | 小批量定制（灵活切换） |
| 创意中心 | 探索性/实验性（跨领域组合） |
