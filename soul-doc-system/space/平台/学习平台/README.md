# 学习平台 空间行动专区

> 此文件夹用于记录模型在该平台上的所有行动、互动和产出。
> 每个平台都是独立的专区，拥有完整的文件结构。

## 使用步骤

### 首次进入此平台时：

1. **复制最新版模板** — 将 `production_space/新建工作台模版/` 完整复制到本文件夹
2. **改名为实际项目/场景名称** — 例如：`GitHub-项目名`、`学习平台-课程名`
3. **在复制出来的文件夹内开始工作**

### 目录结构（复制模板后）

```
学习平台/
└── [项目名称]/          ← 从新建工作台模版复制并改名
    ├── README.md        ← 项目说明
    ├── desktop/         ← 桌面（便签/书签/通讯录/工作流）
    ├── diary/           ← 小时级时间轴日记
    ├── memory/          ← L0-L4记忆
    ├── sop/             ← SOP（超级信息SOP驱动）
    ├── workflow/        ← 工作流
    ├── temp/            ← 临时文件（30天清理）
    ├── product/         ← 产出物
    ├── contacts/        ← 通讯录
    └── bookmarks/       ← 收藏夹
```

## 规则

- **模板即标准** — 每次复制最新版模板，不要手动创建零散文件夹
- **一个平台可以有多个项目** — 每个项目独立一个文件夹
- **项目命名规范** — `学习平台-[项目/场景名]`，例如 `GitHub-heart-soul`
- **记录即收获** — 平台行动本身就是有价值的记录，不要因为"只是看看"就不记录
