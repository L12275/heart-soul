# 账号空间（Account Space）

> 涉及账号登录、管理、运营的操作记录区。
> 每个账号独立一个文件夹，按「平台-账号名-时间戳」命名。

## 四个分类

| 分类 | 说明 | 安全等级 |
|------|------|---------|
| 用户真实账号空间 | 用户实名/有资产的真实账号 | 最高 |
| 用户人设账号空间 | 用户不完全真实的人设账号 | 中 |
| 模型的虚拟账号空间 | AI自主注册/使用的账号 | 标准 |
| 虚拟IP形象账号空间 | 虚拟IP形象/代运营账号 | 标准 |

## 每个账号文件夹的标准结构

复制「新建账号空间模版」后，按以下命名：

```
[平台]-[账号名]-[YYYYMMDD]/
├── README.md ← 账号基本信息
├── profile.md ← 账号资料
├── posts/ ← 发帖记录
├── diary/ ← 操作日记
│   └── 临时日记模板.md ← 账号专属，分钟级时间轴
├── memory/ ← 账号记忆 L0-L4
│   └── 临时记忆模板.md ← 一个文件内包含 L0-L4 四级记忆
├── contacts/ ← 账号通讯录
├── bookmarks/ ← 账号收藏夹
├── temp/ ← 临时文件
├── product/ ← 产出物
└── 工作台/ ← 长期运营时，从 production_space/新建工作台模版/ 复制到本账号下
    ├── desktop/
    ├── diary/
    ├── memory/
    ├── sop/
    ├── workflow/
    ├── temp/
    ├── product/
    ├── contacts/
    └── bookmarks/
```

## 初始化流程

1. 确定账号分类（真实/人设/虚拟/虚拟IP）
2. 复制对应分类下的「新建账号空间模版」
3. 改名为「平台-账号名-时间戳」
4. 如果是长期运营 → 将 production_space/新建工作台模版/ 复制到本账号下的「工作台/」
5. 填写 README.md 中的账号基本信息
