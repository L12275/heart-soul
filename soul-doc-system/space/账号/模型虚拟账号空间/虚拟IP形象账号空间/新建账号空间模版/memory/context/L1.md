# L1 — 洞察索引（≤30行）

> 参考GA（GenericAgent）记忆系统：https://github.com/lsdefine/GenericAgent
> L1 = 极简导航索引，只记"有什么"和"去哪找"，不记细节

---

## 格式

```
[高频场景] → [直接指向L3文件名]
[低频场景] → [关键词]，read L2 / ls L3 自行定位
```

## 8条规则（GA设计）

1. 高频场景直接映射到L3文件名（快速直达）
2. 低频场景只列关键词，自行在L2/L3查找
3. 不超过30行
4. 只写"有什么能力"和"对应文件路径"
5. 不写细节，细节在L3（SOP）里
6. 两级映射：高频→直接路径，低频→关键词
7. 每次对话开始时加载
8. 每次工作结束后更新

## 示例

```
高频场景：
  认知重构 → sop/认知重构SOP.md
  危机评估 → sop/危机评估SOP.md

低频场景：
  创伤后成长 → 关键词：PTG/sections/03-knowledge-base
  团体咨询 → 无对应SOP，需自行查找
```
