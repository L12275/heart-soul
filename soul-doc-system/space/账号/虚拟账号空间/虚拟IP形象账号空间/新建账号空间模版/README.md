# [账号名称] 账号空间

> 此文件夹记录单个账号的所有操作、更新和记忆。
> 每个账号独立一个文件夹，按「平台-账号名-时间戳」命名。

## 文件夹结构

```
[账号名称]/
├── README.md          ← 本文件（账号基本信息）
├── profile.md         ← 账号资料（头像、简介、昵称等）
├── posts/             ← 发帖记录
│   └── YYYY-MM-DD_帖名.md
├── diary/             ← 账号操作日记（小时级时间轴）
│   └── 年/月/日/[日期].md
├── memory/            ← 账号记忆（L0-L4）
│   ├── README.md
│   └── context/
│       ├── L0.md
│       ├── L1.md
│       ├── L2.md
│       ├── L3.md
│       └── L4.md
├── contacts/          ← 账号通讯录
├── bookmarks/         ← 账号收藏夹
├── temp/              ← 临时文件（30天清理）
├── product/           ← 账号产出物
│   ├── 研究类/
│   ├── 执行类/
│   └── 文档类/
└── 工作台/            ← 长期运营账号才需要（从新建工作台模版复制）
    ├── desktop/
    ├── diary/
    ├── memory/
    ├── sop/
    ├── workflow/
    ├── temp/
    ├── product/
    ├── contacts/
    └── bookmarks/
```

## 账号基本信息

- **平台：** [平台名称]
- **账号名：** [账号名称]
- **账号类型：** [真实账号/人设账号/虚拟账号/虚拟IP账号]
- **创建时间：** [YYYY-MM-DD]
- **运营模式：** [长期运营/短期操作]
- **安全等级：** [高/中/低]

## 使用规则

1. **短期操作** — 只用本文件夹，不需要复制工作台
2. **长期运营** — 必须复制「新建工作台模版」到「工作台/」文件夹
3. **每次操作后记录** — 操作内容写入 diary/ 和 memory/
4. **安全第一** — 真实账号严禁异常操作，防止封号和法律风险
