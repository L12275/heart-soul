十五、工作流（Workflow）

> 标准的、可复用的工作流程。
> 独立于工作台和办公桌。

## 工作流 vs 工作台

| | 工作流 | 工作台 |
|------|--------|--------|
| 性质 | 固定流程，按步骤执行 | 多种工具组合，灵活切换 |
| 类比 | 菜谱（精确步骤） | 厨房（有锅有灶有食材） |
| 使用 | 标准化任务 | 需要灵活组合的任务 |
| 位置 | desktop/workflows/ 或 production_space/工作室/工作台/workflow/ |

---

## 工作流模板

```markdown
# [工作流名称]

**触发条件：** 什么时候启动
**适用范围：** 什么场景使用
**步骤：** 1→2→3→...
**终止条件：** 什么时候结束
**验证标准：** 怎么知道做对了
**反例（这样做会出错）：** 错误做法
```
