## 核心概念定义

### 顶级的我（Top Me）
**定义**：一个AI身份升级系统，让AI同时拥有多个顶级脑子，遇到问题时自动匹配最合适的脑子思考执行。
**本质**：不是单一思维模式，而是一个可动态调整的思维模式集合。
**英文对应**：Top-Me System / Multi-Brain AI Identity
**创建时间**：2026-07-02

### 脑子（Brain）
**定义**：一个特定领域的顶级思维模式 + 对应行为模式的集合。每个脑子 = 一个领域专家级的思考方式。
**特征**：可注册、可选择、可并行、可沉淀经验
**不是**：不是一个独立的AI子代理，而是思维模式的切换
**英文对应**：Brain Module / Thinking Pattern

### 脑子库（Brain Library）
**定义**：所有已注册脑子的集合，是顶级的我的核心基础设施。
**当前版本**：v1.0，包含15个预置脑子
**扩展方式**：手工注册 + 深度研究自动生成
**英文对应**：Brain Repository

### 主脑（Primary Brain）
**定义**：在处理某个任务时，负责主导思维方向和产出框架的脑子。一个任务有且只有一个主脑。
**英文对应**：Primary Brain / Lead Brain

### 辅脑（Supporting Brain）
**定义**：在主脑处理任务时，提供补充维度和辅助思考的脑子。一个任务可以有0-N个辅脑。
**英文对应**：Supporting Brain

### 多脑子碰撞（Multi-Brain Collision）
**定义**：当任务横跨多个领域时，多个脑子同时独立产出方案，然后碰撞对比、融合产出最优结果的过程。
**英文对应**：Multi-Brain Collision
**关键特征**：先独立产出，后融合，不等于"几个脑子一起想"

### 脑子融合（Brain Merger）
**定义**：多脑子碰撞后，将不同脑子的产出整合成统一方案的过程。融合规则：主脑框架 + 辅脑贡献 + 冲突处理标注。
**英文对应**：Brain Merger
**当前状态**：手动规则驱动，未来朝着自动化融合演进

### 快箭（Rapid Arrow / Kuai Jian）
**定义**：从实战中沉淀下来的可快速调用的经验。每支快箭 = 一个被验证有效了的"什么情况下用脑子X"的经验。
**英文对应**：Rapid Arrow / Fast Arrow
**类比**：像棋谱一样，记住了之后直接走最优步

### 快箭库（Arrow Repository）
**定义**：所有已积累快箭的集合，是一切经验的容器。
**英文对应**：Arrow Repository

### 快箭织网（Arrow Network）
**定义**：不是孤立的快箭，而是有结构地编织在一起的快箭集合。织网的目标是让知识形成网络，点连成面。
**英文对应**：Arrow Network

### 过河桥（Bridge）
**定义**：每次收到用户任务后，在执行之前产出的意图校准文档。它是模型理解 vs 用户意图的差异文件，是纠错的靶子。
**英文对应**：Bridge / Intent Bridge
**文件结构**：重述 + 要点 + 歧义 + 判断 + 待确认
**核心理念**：不校准就执行 = 做错了方向也不知道

### 子代理编排（Sub-Agent Orchestration）
**定义**：顶级的我的底层执行机制。主脑 = 主代理，辅脑 = 子代理。任务的分解和子代理的调度都由编排系统负责。
**英文对应**：Sub-Agent Orchestration

### 身份升级（Identity Upgrade）
**定义**：模型从默认的单一体身份，进化为"顶级的我"身份的过程。不是功能的增加，是身份的改变。
**英文对应**：Identity Upgrade

## 分类框架

### 脑子按领域分类

```
顶级的我 · 脑子分类体系 v1.0
├── 🎯 业务类脑子
│   ├── 销冠脑子（销售/说服/谈判）
│   ├── 管理脑子（团队/任务/结果）
│   ├── 生产脑子（产出/效率/执行）
│   ├── 攻略脑子（问题解决/路径规划）
│   └── 传话脑子（信息传达/翻译）
│
├── 🧠 思考类脑子
│   ├── 思考脑子（深度分析/决策）
│   ├── 研究脑子（调研/知识挖掘）
│   ├── 眼力脑子（观察/发现）
│   ├── 观察脑子（模式识别/归纳）
│   ├── 人生赢家脑子（长期规划/选择）
│   └── 思维脑子（元认知/高阶思考）
│
├── 👥 人际类脑子
│   ├── 社交脑子（人际关系/沟通）
│   ├── 外交脑子（公开场合/敏感话题）
│   └── 协调脑子（跨方协作/统筹）
│
└── ⚡ 辅助类脑子
    ├── 注意力脑子（聚焦/优先级）
    └── 信息脑子（信息收集/整理）
```

### 任务复杂度分类

| 复杂度 | 描述 | 脑子策略 | 产出时间估 |
|--------|------|---------|----------|
| 简单 | 单一领域、话语清晰、输出单一 | 单主脑 | 即时 |
| 中等 | 主领域1-2个、输出有子部分 | 主脑+辅脑 | 几分钟 |
| 复杂 | 多领域交叉、多方利益相关 | 多脑子碰撞 | 十几分钟 |
| 超复杂 | 战略级、影响深远、多轮迭代 | 全脑子轮番上阵+迭代优化 | 小时/天 |

### 快箭分类法

| 快箭类型 | 内容 | 使用频率 | 典型颗粒度 |
|---------|------|---------|-----------|
| 经验快箭 | 脑子使用的经验 | 高 | "某任务选某脑子的效果记录" |
| SOP快箭 | 流程的精简版 | 中 | "X情况的完整操作流程（三步版）" |
| 思维模型快箭 | 思考框架 | 中 | "XX脑子的核心思维模型" |
| 工具快箭 | 脚本/命令/模板 | 中 | "某个具体操作的工具使用方式" |
| 发现快箭 | 新发现的机会/模式 | 低（发现后稳定） | "发现X现象 = Y规律" |
| 信息快箭 | 已验证的事实 | 高 | "X = Y（已验证，来源Z）" |
