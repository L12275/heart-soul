衣柜

> **存放智能体的"衣服"——经过验证的智能体配置模板。**
> 衣柜 = 模板仓库，衣服 = 智能体配置。

## 规则

- 模板必须是已经验证过的配置
- 每个模板包含 README.md（说明） + 配置文件（.json）
- 衣柜的内容不直接运行——复制到大厅再运行
- 有价值的大厅修改 → 提炼为新模板 → 存入衣柜

## 已有模板

- `通用智能体/` — 通用任务执行智能体（一件通用马甲）

## 添加新模板流程

1. 在大厅验证配置
2. 验证通过 → 提炼为模板
3. 存入衣柜，带 README.md 说明

## 灵魂继承

- 通用区衣柜的智能体 → 继承**主灵魂文档**
- 子角色区衣柜的智能体 → 继承**子角色灵魂文档**
