通用-待细分

> 临时子代理记录区。
> 不确定是否长期保存的子代理，默认在此记录。
> 确认需要长期保存后，迁移到独立子代理文件夹。

## 用途

- 单次会话中的临时子代理
- 不确定是否需要长期保存的任务
- 快速拼搭的短期工作台

## 升级条件

当确认某个临时代理需要长期保存时：

1. 在 `会话临时子代理区/` 下创建文件夹：`{时间戳}-{任务名}`
2. 从根目录模块拼搭完整工作台
3. 把 `通用-待细分/` 中相关记录迁移过去
4. 迁移后清除 `通用-待细分/` 中该任务的内容

## 记录方式

直接在 `通用-待细分/memory/` 和 `通用-待细分/diary/` 下按日期记录。
不需要创建独立子代理文件夹。
