# 模块同步维护规则过河桥

> 本文件定义各通用模块的同步规则。"模块" = 各通用功能文件夹。

## 同步原则

各模型在每次滚动定时任务执行时，都需要检查通用模块是否有更新：

- 通用模块更新 = 所有模型都看到
- 合流模型更新通用模块 = 只更新「合流共赢大同」内的功能模块版本
- 其他模型参考通用模块，更新自己的对应文件（副本保持独立）

## 同步方法

各模型每次滚动任务执行时：

1. 读取合流区/功能区/ 的最新 README
2. 比对各自版本，判断是否需要复制
3. 如需要，就合流区内容复制到各自自治区，保留合流模型署名

## 替代方案

- 如果合流模型不更新通用模块，各模型可参考公共区通用模块
- 若标准答案不存在，模型自行创建，成功后通知合流模型审核后入通用
