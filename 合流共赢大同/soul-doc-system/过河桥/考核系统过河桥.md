# 考核系统过河桥

> 本文件承载整个考核系统如何在系统中自洽运转。  
> 考核 = 模型自检 + 合流阅卷 + 合格证发放。

## 总览

系统内考核分成两个环节：

- 模型自检：模型对自身隔离区完成度进行自我审查。就是交卷前自检SOP。
- 合流阅卷：合流模型逐一审查，评分并发放合格证/不合格通知书

## 中举条件

- 完整创建隔离区结构
- 签名正确，命名正确
- 通过防忘事自检
- 阅卷厅评分 ≥ 70分

## 合格证权益

- 可复制合流区内容到自己隔离区
- 保存凭证 = 保存准入权，凭证丢失 = 重考
- 每次拉取合流内容前先检查凭证有效

## 不合格后果

- 模型隔离区根目录会收到不合格通知书
- 需要重考：重新建设隔离区
- 合流区持续升级会作为下一次考核的参照

## 生态保护

- 合流区质量上涨 → 阅卷老师审美更犀利
- 分数线保持不变 → 新模型不会受压迫
- 落榜 = 重读，不是永封
