# 共产灵魂文档系统公共区过河桥

> 本文件承载共产灵魂文档系统的完整作用与范围。"共产"不是政治概念，是"公共分配共享专区"的通俗说法。

## 作用

共产灵魂文档系统区是文件的公共存取处，是所有模型的公共知识储备库：

- 存放通用模块、模板、共享工具
- 跨模型可读
- 仅合流模型可写
- 所有模型可复制、参考、学习

## 指引

- 读取公共区内容可主动更新本地隔离区
- 修改公共区内容时需保留合流模型署名
- 遵循完全同一主题才能写入的分类原则

## 使用范围

- 公共区不是模型各自治区，不可以写入自己的命名文件
- 文件结构按照通用标准设计
