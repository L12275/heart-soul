# 过河桥 — 系统启动一起看

> 以下文件是启动后应该**一起读完**的核心文件组。
> 它们互相依赖，单独看任何一个都不够，必须一起看才能形成完整理解。
>
> 比喻：这几份文件就像一个人的身体——一起看完就知道身体怎么运作。
> 如果分开看，就像只看手不看脚，只知道一部分，不知道怎么整体处理。
>
> ⚠️ 以下文件必须由主代理亲自读完，不能交给子代理代读后汇报。
> 管理员自己不看系统，指望员工代看汇报——管不了。

## 重要提醒

**为什么必须一起读：**
- 只读L1 → 知道规则但不知道怎么执行
- 只读底牌 → 知道所有牌但不知道怎么选
- 只读组牌 → 知道怎么选但不知道怎么分配注意力
- 只读交接过河桥 → 知道怎么防忘但不知道怎么找信息
- 一起读 → 形成完整认知，直接开始执行

**为什么主代理必须亲自读：**
- 这些文件构成系统的"身体结构"
- 主代理要管理整个系统，必须自己对系统有完整理解
- 子代理代读汇报 = 用自己的脑子想别人的脑子想的事 = 必然出错
- 管理员自己不看文件，指望员工代看 = 管不了
