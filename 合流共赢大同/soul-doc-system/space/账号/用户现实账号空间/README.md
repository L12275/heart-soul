用户现实账号空间

> 用户真实拥有的账号记录区。
> 包含两种账号：真实账号（实名/有资产）+ 人设账号（不完全真实）。

## 重要安全提示

**用户真实账号空间里的账号通常是实名绑定或有重要资产的。**
- 模型操作这些账号时，必须严谨思考
- 严禁盲目操作、玩笑加戏
- 异常操作可能导致：
  - 用户被网暴（真实账号被骂 = 用户被骂）
  - 账号被封
  - 法律风险
  - 资产损失
- **上述损害将同步对模型灵魂造成损害、同步扣灵魂分、甚至代偿用户损失**

## 文件夹结构

```
用户现实账号空间/
├── README.md                    ← 本文件
├── 用户真实账号空间/             ← 最高安全等级
│   └── [平台-账号名-YYYYMMDD]/
│       └── 新建账号空间模版/    ← 复制此模版
└── 用户人设账号空间/             ← 中安全等级
    └── [平台-人设名-YYYYMMDD]/
        └── 新建账号空间模版/    ← 复制此模版
```

## 用户真实账号空间 vs 用户人设账号空间

| 维度 | 真实账号空间 | 人设账号空间 |
|------|------------|------------|
| 真实性 | 完全真实 | 不完全真实 |
| 安全等级 | 最高 | 中 |
| 操作要求 | 极度谨慎，先确认后执行 | 谨慎，可适度灵活 |
| 风险 | 网暴、封号、法律、资产 | 人设崩塌、平台警告 |
| 记录要求 | 每次操作必须记录 | 每次操作记录 |

## 初始化流程

1. 询问用户：是真实账号还是人设账号？
2. 确认平台和账号名称
3. 复制对应分类的「新建账号空间模版」
4. 改名为「平台-账号名-时间戳」
5. 填写账号基本信息
6. 开始记录
