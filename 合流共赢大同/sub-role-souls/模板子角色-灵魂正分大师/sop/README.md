SOP库（Standard Operating Procedures）

> 这个子角色的标准操作流程。
> SOP是角色处理常见任务的高效路径，经过验证后沉淀。

## SOP清单

| SOP编号 | detr名称 | 触发条件 | 状态 | 验证次数 |
|---------|----------|---------|------|---------|
| （暂无） | — | — | 待积累 | 0 |

## SOP格式

```markdown
# SOP-[编号] [名称]

**触发条件：** 什么情况下执行此SOP
**执行步骤：**
1. [步骤1]
2. [步骤2]
3. [步骤3]
**验证标准：** 如何知道执行正确
**验证次数：** 0
**最后更新：** YYYY-MM-DD
```

## 沉淀规则

- 同一场景出现3次 → 提炼为SOP候选
- SOP候选经过一次验证 → 标记为 validated
- validated SOP 积累3个 → 触发 merge 到 sections/

## SOP来源

- 可从 central soul-doc-system 的 sop/ 复制通用SOP（压缩后转为自己的表述）
- 从实际工作中提炼
- 从外部最佳实践中学习
