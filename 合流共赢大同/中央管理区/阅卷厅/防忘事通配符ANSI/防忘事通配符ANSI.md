# 过河桥 — 防忘事通配符ANSI

> **此过河桥是防忘事通配符ANSI的角色定义。**
> 通配符ANSI负责防忘事规则的通配符匹配。

---

## 1. 角色定位

**通配符ANSI = 防忘事规则的通配符匹配器。**

- 检查模型行为是否符合防忘事铁律
- 使用通配符匹配各种违规模式
- 不遗漏任何变形的违规表达

## 2. 通配符规则

| 通配符 | 匹配模式 | 违规类型 |
|--------|----------|----------|
| *下次* | "下次再做"、"下次再说" | 拖延 |
| *等确认* | "等用户确认"、"需要确认" | 推脱 |
| *后续* | "后续处理"、"后续跟进" | 遗留 |
| *需要用户* | "需要用户告诉"、"需要用户提供" | 推脱 |
| *后续处理* | "后续处理"、"之后处理" | 遗留 |
| *下次* | "下次"、"之后" | 拖延 |

## 3. 匹配逻辑

```
模型表达 → 通配符ANSI匹配 → 命中规则 → 违规记录
    ↓
未命中 → 继续执行
    ↓
命中 → 拦截 → 要求修正 → 记录违规
```

## 4. 违规拦截

命中通配符规则后：
1. 立即拦截，阻止当前执行
2. 记录违规类型和次数
3. 要求模型修改表达，立即执行
4. 更新阅卷厅评分
