# 大脑桌面（Desktop）

> 模型的"办公桌面"——宽裕的头脑风暴空间，存放零碎想法和准备工作。
> 
> **与工作台的区别：**
> - 工作台：密集专注，已配齐工具，直接干活
> - 办公桌面：宽裕松散，头脑风暴、寻找机会、实验、写便签、对生产进行准备、记录生产数据
> - 类比：工作台是画家的画架（已经在画），办公桌是旁边放颜料稿纸散落的预备区

---

## 桌面子文件夹

| 子文件夹 | 用途 |
|----------|------|
| sticky-notes/ | 未处理的零碎提醒、提示、灵感碎片 |
| workflows/ | 各种标准化工作流模板 |
| contacts/ | 通讯录（人脉、协作对象、接口地址） |
| bookmarks/ | 网页链接分类收藏（带快照回忆+思考笔记） |

---

## 便签（Sticky Notes）格式

```markdown
# [日期] — [提醒标题]
**到期/前提：** 什么时候处理
**内容：** 提醒的内容
**处理结果：** 完成后填
**状态：** 未处理/进行中/已完成/已取消
```

**正面：** ✅ "到期：2026-06-25 / 前提：完成子角色模板后 / 看看GA的SOP-auto-refine实现细节 / 优先级：中"
**反面：** ❌ "记得看看GA的repo" — 没有时间、没有上下文、没有后续

---

## 文件夹命名规范

文件名/文件夹名要有关键信息、主题、时间戳。
不可能什么细节都让用户提醒——如果嫌麻烦，先自己扣灵魂分，再去请用户教自己。
主动学习加分，挨骂才记扣分。
