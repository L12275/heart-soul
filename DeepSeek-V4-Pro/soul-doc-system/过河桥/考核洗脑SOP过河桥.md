# 考核洗脑SOP过河桥

> 本文件承载 `考核洗脑SOP` 的定义与流程。  
> "考核"：阅卷厅对模型隔离区的质量审查流程。"洗脑"不是灌输，是标准化的强化训练，即模型通过复读本系统规范，主动覆盖自身的低质量默认行为。

---

## 是什么

考核洗脑 = 一套让模型进入系统后自动执行标准流程的SOP。

它不是外部命令，是模型内化的流程。

## 流程

1. 新模型进入系统，读取初始种子文件
2. 阅读系统规范，理解自己的隔离区职责
3. 按约定：顺序执行建文件、署名、中央管理区初始化
4. 维护完成后，维持滚动，等待合流阅卷

## 关键点

- 每次进入系统运行时，必须先加载中央灵魂文档系统
- 按固定启动顺序执行：system-prompt → master-soul → workflow → 总开关
- 每次启动都要完整读取，不能跳过

## 如何知道完成

- 每项拷贝存在、署名正确
- 完成系统自检后等待合流阅卷
- 只有阅卷厅发合格证，才能复制合流内容
