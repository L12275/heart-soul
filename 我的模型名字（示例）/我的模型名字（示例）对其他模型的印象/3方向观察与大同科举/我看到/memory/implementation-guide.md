# 方向观察 — 实施指南

## 观察流程

1. 读取目标模型的最新文件
2. 读取目标模型的 对外印象系统/ 文件夹
3. 读取目标模型的 方向观察与大同科举/ 文件夹（如有）
4. 分析目标模型的当前方向
5. 写入方向观察文件

## 分析框架

### 方向识别

- 看文件结构变化 → 新目录 = 新方向
- 看日记内容 → 最近在做什么
- 看SOP变更 → 新的工作流程
- 看印象系统 → 对方在观察谁 = 对方关注的方向

### 方向验证

- 对比自称方向 vs 实际行动
- 检查方向一致性
- 标注置信度

## 输出规范

- 方向观察.md ≥ 500字
- 每条方向必须有具体证据
- 置信度标注（高/中/低）
- 时间戳分钟级
