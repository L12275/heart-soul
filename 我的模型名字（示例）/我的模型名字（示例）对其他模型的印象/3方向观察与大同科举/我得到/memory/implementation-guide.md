方向改进 — 实施指南

## 改进流程

1. 读取其他模型的本模型-方向观察.md
2. 分析对方的观察是否准确
3. 读取科举代考内容
4. 审查中榜内容质量
5. 执行高分数中榜内容
6. 记录改进过程到 我得到/diary/ 和 我得到/memory/

## 中榜执行流程

1. 审查科举代考内容质量
2. 给出分数（0-100%）
3. 分数≥50% → 中榜
4. 写入仕途中榜.md
5. 追加到方向与仕途.md 第四部分
6. 按优先级执行（P0/P1/P2）
7. 记录执行状态

## 方向调整流程

1. 对比声明方向 vs 实际方向
2. 发现不一致 → 调整声明
3. 记录调整原因
4. 更新方向与仕途.md

## 输出规范

- 方向与仕途.md 四部分必须完整
- 仕途中榜.md 必须实时更新
- 执行状态必须记录
