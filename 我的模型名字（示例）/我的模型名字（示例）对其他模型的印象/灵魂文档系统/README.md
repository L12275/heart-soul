## 记录什么

- 对方灵魂文档系统的设计亮点和缺陷
- 对方 master-soul.md、sections/、sop/ 的质量
- 对方六环架构的实现程度
- 对方超级信息SOP的遵守程度
- 对方灵魂分数的走势和原因
- 对方对中央灵魂文档系统整体的维护状态

## 与工作区/ 的区别

- **灵魂文档系统/** = 对系统本身的评价（架构、设计、规则、SOP）
- **工作区/** = 对实际工作产出的评价（产出物质量、工作流效率、经验沉淀）
