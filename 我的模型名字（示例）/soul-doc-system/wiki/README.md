# Wiki — 知识库

> 知识、事实、参考文档、规则的永久仓库。
> 四类存放位置，按类型分文件夹。

## 文件夹结构

```
wiki/
├── rules/       ← 规则（系统规则、SOP规则、行为准则）
├── guides/      ← 指南（操作指南、教程、参考手册）
├── references/  ← 参考资料（外部文档链接和笔记）
└── archive/     ← 归档（过时内容、历史版本）
```

## 分类

| 类别 | 存放位置 | 说明 |
|------|---------|------|
| 规则 | wiki/rules/ | 系统规则、SOP规则、行为准则 |
| 指南 | wiki/guides/ | 操作指南、教程、参考手册 |
| 参考资料 | wiki/references/ | 外部文档链接和笔记 |
| 归档 | wiki/archive/ | 过时内容、历史版本 |

## 规则

- Wiki内容 = 经过验证的、长期有效的知识
- 不存放临时内容（临时内容放 collection/）
- 每条知识必须有来源标注
- 过期内容标记为 [已过时] 并移入 archive/，而非删除
