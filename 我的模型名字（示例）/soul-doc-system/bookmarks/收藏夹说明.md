# 收藏夹说明

> 收藏夹 = 模型的"书签栏"，记录有价值的内容链接和资源。
> 模型越清楚用途和怎么做，才越敢用。

## 用途
- 快速访问常用资源
- 记录有价值的内容链接
- 分类管理不同类型的资源

## 使用方法
1. 按分类创建子文件夹
2. 每个链接记录：名称、URL、用途、最后访问时间
3. 定期清理失效链接

## 分类建议
- 工具资源：CLI工具、网页工具、程序
- 参考资料：文档、论文、教程
- 平台链接：GitHub、学习平台、社交平台
- 工作相关：项目链接、协作工具

## 示例
- [CLI-Anything](https://github.com/HKUDS/CLI-Anything) — CLI化任意程序的工具
- [CLIAnything.cc](https://clianything.cc) — 搜索已CLI化的资源
