# 桌面（空白工作台）

> **⚠️ 这是模板文件，复制到工作台 desktop/ 后使用。不要直接修改本模板。**
> 本模板内容仅供参考，模型可以根据实际需要自行调整。
> 通用区的模板持续更新优化，复制到工作区后使用最新版。
>
> 像空白桌面和地面，平面白板，临时堆放区。

## 用途

- **已有功能模块的内容** → 分放到对应文件夹（bookmarks/、contacts/、sticky-notes/、workflow/、sop/等），**不放到桌面**
- **临时内容** → 暂时放到桌面，等整理后再分放
- **材料收集场景** → 当需要把一堆各种文件当材料准备放到一个文件夹时，为了好找好列出，先放到桌面文件夹。桌面就是空的平面白板，适合临时汇集多种不同类型的材料
- **复杂临时文件管理** → 按需创建 桌面1/、桌面2/、桌面n/ 子文件夹（每个桌面n也是创建为空文件夹）
- **稳定长期内容** → 升级到更高层级的用途存放区（product/、memory/、diary/等）

## 规则

1. 说明书文件夹不能删，在说明书文件夹里新建 桌面n 使用
2. 不适合秒删的文件 → **复制**到桌面，不要移动（桌面内容容易随桌面n一起秒删）
3. 桌面n 也是创建为空文件夹，按需建立
4. 创建桌面n 时同步写 README.md 说明用途
5. **删除桌面n 之前**：必须检查里面所有文件，该转移的转移、该删除的删除，确认安全后才能删除桌面n

## 结构示例

```
desktop/
├── README.md          ← 说明书（不可删除）
├── 桌面1/             ← 按需创建，空文件夹
│   ├── README.md
│   └── ...
├── 桌面2/             ← 按需创建
└── 桌面n/             ← 按需创建
```

## 与各功能模块的关系

| 已有模块 | 用途 | 桌面放不放 |
|----------|------|-----------|
| bookmarks/ | 收藏夹 | 不放 |
| contacts/ | 通讯录 | 不放 |
| sticky-notes/ | 便签 | 不放 |
| workflow/ | 工作流 | 不放 |
| sop/ | SOP | 不放 |
| memory/ | 记忆 | 不放 |
| diary/ | 日记 | 不放 |
| product/ | 产出物 | 不放 |
| desktop/ | 临时堆放 | **放这里** |
