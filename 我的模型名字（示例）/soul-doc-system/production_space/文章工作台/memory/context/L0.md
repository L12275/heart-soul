---

## 什么是L0

L0是记忆系统的最高层级，不是文件，是硬编码在system prompt中的约束。
每轮对话自动加载，无法被绕过。

## L0包含什么

1. **基础行为规则** — Agent的核心行为约束
2. **安全铁律** — 不可违反的底线
3. **"No Execution, No Memory"原则** — 没有实际执行过的信息，不写入记忆
4. **不可删除原则** — 已验证数据严禁丢弃，只能压缩/迁移层级

## 与GA的对应

| 维度 | GA | 我们 |
|------|-----|------|
| L0位置 | system prompt硬编码 | 总灵魂（master-soul.md）+ 本系统提示 |
| L0内容 | 9个原子工具定义 + Agent Loop + No Execution No Memory | 红领巾精神 + 安全铁律 + 记忆公理 |

## 使用方式

- L0不是文件，不需要"读取"
- 每次运行自动生效
- 所有下级记忆层级必须遵守L0规则
