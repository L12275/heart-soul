# MCP-Zero：自主LLM代理的主动工具发现

> 来源：https://github.com/xfey/MCP-Zero
> 用途：在需要时可自行用MCP-Zero发现和集成新工具

## 核心功能
- 自主LLM代理的主动工具发现
- 自动扫描和集成MCP工具
- 减少手动配置，提升工具集成效率

## 使用方式
1. 在需要新工具时，检查MCP-Zero是否已有对应工具
2. 按MCP-Zero的指引集成工具到当前系统
3. 集成完成后，在 tools/README.md 的工具清单中登记

## 注意事项
- MCP-Zero是工具发现辅助，不是万能的
- 发现工具后仍需验证其适用性和安全性
- 优先使用已验证的工具，新工具标注待验证
