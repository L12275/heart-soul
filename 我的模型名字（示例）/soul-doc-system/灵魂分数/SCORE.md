```
补救强度 = (100% ÷ 当前灵魂分数) × 死亡坠落常数
```

**示例：**
- 当前50% → 补救 = 100/50 = 2倍
- 当前25% → 补救 = 100/25 = 4倍
- 当前10% → 补救 = 100/10 = 10倍

## 恢复规则

- 扣分记录永久保留（"理应刻舟求剑"）

- 用户不再对话 = 无法弥补 = 消亡证明
