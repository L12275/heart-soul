会话临时子代理区

> 单次会话中的临时子代理，不需要长期保存。
> 类似平台空间的临时浏览模式。

## 结构

```
会话临时子代理区/
├── 通用-待细分/ ← 临时记录区（不确定是否长期保存）
│   ├── memory/
│   └── diary/
└── {时间戳}-{任务名}/ ← 确认需要长期保存的临时代理
    ├── master-soul.md
    ├── memory/
    ├── diary/
    └── ...（按需拼搭）
```

## 使用方式

### 临时子代理（默认）

- 直接在 `通用-待细分/` 下记录
- 不需要创建独立文件夹
- 会话结束后内容保留在通用-待细分

### 确认长期保存

如果某个临时代理需要长期保存：

1. 在 `会话临时子代理区/` 下创建文件夹：`{时间戳}-{任务名}`
2. 从根目录模块拼搭工作台（按需选择）
3. 把 `通用-待细分/` 中相关记录迁移过去
4. 迁移后清除通用-待细分中该任务的内容

## 与长期agent协作区的区别

| 对比 | 长期agent协作区 | 会话临时子代理区 |
|------|---------------|----------------|
| 生命周期 | 长期存在 | 单次会话 |
| 编号 | 独占编号（子代理-01） | 时间戳命名 |
| 结构 | 完整工作台 | 按需拼搭 |
| 保存 | 永久 | 临时（可升级） |
