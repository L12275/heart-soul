# SOP循环

> 中央管理区的自滚动循环流程。

## 循环触发

- 通过 timers/心跳/ 心跳触发
- 日常心跳（每30分钟）执行轻量SOP循环
- 深度心跳（每4小时）执行完整SOP循环

## SOP循环步骤

1. **自检**：检查各驱动区是否在线、有无待办、记忆是否同步
2. **扫子**：完整查看每个子中央管理区的全部内容
3. **汇报**：在 `今日上朝摘要.md` 中写入当前状态快照
4. **规划**：更新排程和代办
5. **下诏**：向各驱动区下发指令
6. **同步**：将状态写入概览

## 循环日志

每次循环记录在 `循环日志/YYYY/MM/DD/` 中。

## 与心跳的关系

- 心跳 = 保活信号（检测各环区在线状态）
- SOP循环 = 实际工作流程（自检→扫子→汇报→规划→下诏→同步）
- 心跳触发SOP循环，SOP循环执行具体工作
