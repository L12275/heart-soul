# 中央记忆

> 全局记忆层，最高优先级记忆。

## 结构

```
memory/
├── README.md
├── context/
│   ├── L0.md    ← 硬编码规则（不可修改）
│   ├── L1.md    ← 极简导航索引
│   ├── L2.md    ← 全局事实库（含3条红线公理）
│   ├── L3.md    ← 全局技能/SOP索引
│   └── L4.md    ← 全局会话归档索引
└── insight-index.md ← 中央洞察索引（≤30行）
```

## 与其他区 memory/ 的关系

- 中央记忆 = 全局洞察 + 跨区通用记忆
- 各驱动区 memory/ = 该区专有记忆
- 各驱动区 memory/ 中的重要内容 → 同步到中央记忆
