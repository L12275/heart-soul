# 模板子角色 — Git Tree

> 子代理工作分支与合流管理。
> Git历史与远程仓库推送。

## 结构

```
git-tree/
├── remotes/           ← 远程仓库配置
├── history/           ← Git历史记录
├── convergence/       ← 合流记录
└── desktop/           ← Git桌面快捷方式
```

## 规则

- 每个子代理 = 一个分支
- 分支命名：`subrole/[名称]/[任务ID]`
- 合流前必须通过质量检查
- 远程推送前必须验证完整性
