# 模板子角色 — 发现

> 子角色主动发现机会、探索、启发、主意、策略、点子的记录。

## 发现记录格式

```markdown
## [发现编号] — [发现标题]
**时间：** YYYY-MM-DD HH:MM
**角色：** [哪个子角色发现的]
**来源：** [在什么任务/对话中发现的]
**发现类型：** 机会/启发/策略/点子/探索发现
**描述：** [发现了什么]
**价值评估：** 高/中/低
**下一步：** [可转化为什么行动]
```

## 发现流转

发现 → 记录 → 评估 → 高价值→任务/SOP / 中价值→backlog / 低价值→归档
