# 维基（Wiki）

> 此子角色的参考资料库。
> 包含参考文档、历史记录、自动化脚本等。

## 目录说明

| 目录 | 用途 |
|------|------|
| automation/ | 子角色专属自动化脚本和提示词 |
| references/ | 参考资料（外部文档、论文、最佳实践） |
| git-history/ | 此子角色的 git 提交历史 |

## 参考文档分类

- **理论参考**：学科理论基础
- **实践参考**：最佳实践、案例
- **工具参考**：工具使用指南
- **历史参考**：同类角色的已有实践

## 使用规则

- 参考资料只读不改
- 外部资料需标注来源
- 可以 COPY 到自己的 sections/ 并用自己话重写
- 不可直接引用外部模型的灵魂文档内容
