## 结构

```
memory/
├── README.md
├── context/
│   └── L0-L4.md    ← L0-L4记忆层级（L0硬编码）

```

## 与子角色灵魂文档系统 memory/ 的区别

- **本 memory/**（子中央管理区）= 子角色身份锚定记忆 + 跨会话通用记忆
- **子角色 soul-doc-system memory/** = 子角色专有记忆（专业技能、领域知识）

## 共享规则

多个子中央管理区之间：
- 子中央管理区的记忆不互相直接写入
- 通过主中央管理区协调共享
- 主中央智能体有权检查和更新子中央管理区的记忆
