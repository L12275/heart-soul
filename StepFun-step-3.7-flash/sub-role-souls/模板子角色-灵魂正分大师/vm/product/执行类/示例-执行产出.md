# [执行类产出示例]

> 执行类产出 = 具体的咨询记录、干预方案、评估报告等
> 特点：有具体步骤、有时间线、有可追踪的结果

## 示例：认知重构干预记录

### 基本信息
- 来访者：A（化名）
- 日期：2026-06-20
- 类型：第3次CBT干预
- 时长：50分钟

### 干预前状态
- GAD-7评分：7/10（中度焦虑）
- 主要思维陷阱：全有或全无（"我必须完美，否则就是失败"）
- 情绪：愤怒(6/10) + 焦虑(7/10)

### 干预步骤

1. **思维识别（10分钟）**
   - 引导来访者记录本周出现自动负性思维的时刻
   - 发现5次"我必须…"模式的思维

2. **证据检验（15分钟）**
   - 支持思维的证据：上次提交报告时犯了错
   - 反对思维的证据：过去3个月成功完成了4个报告
   - 来访者自评：反对证据更强，但"犯错那一刻很痛苦"

3. **替代思维生成（15分钟）**
   - 旧思维：「我必须完美，否则就是失败」
   - 新思维：「我尽力做到最好，犯错是可以修正的过程」
   - 可信度：来访者给新思维打40%（"听起来对，但感觉上还没接受"）

4. **行为实验设计（10分钟）**
   - 本周任务：下次遇到"必须完美"念头时，写下3条替代思维
   - 检查：每天记录焦虑评分变化

### 干预后状态
- GAD-7评分：4/10（下降）
- 主要思维陷阱出现频率：从5次/周降至 预计2-3次
- 情绪：愤怒(3/10) + 焦虑(4/10)

### 反面产出（坏的写法）
```
来访者A，第三次咨询。
来访者有完美主义思维，做了认知重构。
来访者感觉好多了。
```
❌ 错误：没有具体步骤、没有评分变化数字、没有行为实验设计。
"感觉好多了"不等于"有效果"——需要量化。
