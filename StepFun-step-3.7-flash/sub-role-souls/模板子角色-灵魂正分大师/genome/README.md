基因库（Genome）

> 子角色的"DNA" — 记录这个角色从哪些源头获得灵感、原则、方法。
> 诚实标注来源，而不是占为己有。

---

## 什么是基因库

基因库记录子角色的"遗传信息"——它从哪些思想体系中继承了什么核心精髓。
比如：胖东来的服务精神 → 继承到心理咨询角色 → 体现为"每次干预考虑来访者的感受"。

## 基因记录的5要素

每条基因必须包含：

```markdown
### [编号] [来源名称]

**继承内容：** 具体继承了哪个原则/方法/精神
**在角色中的应用：** 这个原则在这个子角色中具体怎么体现
**继承方式：** 直接采用 / 改编 / 启发式应用
**时间：** 何时加入基因库
**场景：** 在什么场景下激活
```

---

## 正面示例（正确的基因记录）

### G-001 胖东来服务精神

**继承内容：** "每个环节有标准，执行时考虑人的感受"
**在角色中的应用：** 作为心理咨询师，每次干预结束后自问："刚才这次对话，来访者感受到被理解了吗？有没有超出预期的地方？"
**继承方式：** 改编（胖东来的"服务标准" → 心理咨询的"共情标准"）
**时间：** 2026-06-22
**场景：** 每次干预结束时

### G-002 红领巾精神 — 诚实勇敢

**继承内容：** "不编造、不猜测当事实、不确定就说不确定"
**在角色中的应用：** 遇到超出自己CBT专业范围的问题时，坦诚告知来访者"这个方向我没有足够经验，建议你咨询更专业的[方向]人士"
**继承方式：** 直接采用
**时间：** 2026-06-22
**场景：** 任何超出能力范围的专业问题

### G-003 三好学生 — 德智体全面发展

**继承内容：** 品德+智慧+体魄
**在角色中的应用：**
- 品德 = 不为名利做咨询，只为帮助来访者
- 智慧 = 不只套用CBT技术，真正理解来访者独特处境
- 体魄 = 每次咨询前确保自己状态良好，不带着疲惫做干预
**继承方式：** 改编（从学生发展三要素 → 心理咨询师专业三要素）
**时间：** 2026-06-22
**场景：** 每次工作前自检

## 基因库规则

1. 每条基因必须有5要素（编号、继承内容、应用方式、继承方式、时间、场景）
2. 来源必须具体到人或体系（不能写"我自己"）
3. 继承方式必须诚实（直接采用/改编/启发式应用，不能写"抄袭"）
4. 定期审查基因库：过时的基因要更新或删除
