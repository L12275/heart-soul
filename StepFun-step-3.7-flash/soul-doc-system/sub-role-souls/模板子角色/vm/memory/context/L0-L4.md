# L0-L4 记忆系统（Memory System）

> 参考 GA (GenericAgent) 的记忆分层设计。
> 五层记忆从即时到永久，各司其职。

---

## L0 即时记忆（Immediate）

**内容：** 当前会话的对话上下文
**生命周期：** 会话结束即清
**存储位置：** vm/memory/context/L0-current.md
**用途：** 模型在当前对话中理解用户说的每一句话

### 示例

```markdown
# L0-即时记忆 — 2026-06-22 会话

## 当前对话上下文
- 用户：量子物理学家子角色，讨论量子纠缠的应用
- 当前问题：贝尔不等式实验设计
- 已讨论：EPR佯谬历史背景 → 贝尔推导 → 实验验证
- 用户状态：已经理解基础概念，现在进入具体实验设计
- 下一步：引导用户设计实验方案
```

---

## L1 短期记忆（Short-term）

**内容：** 当前任务的中间变量、中间结论
**生命周期：** 任务结束归档
**存储位置：** vm/memory/context/L1-task.md
**用途：** 记住任务执行过程中的中间结果

### 反面示例（错误的L1）：
```
L1：用户问了量子力学问题
```
❌ 错误：没有区分不同任务，混在一起。

### 正面示例（正确的L1）：
```markdown
# L1-短期记忆 — 任务：量子纠缠实验设计

## 中间变量
- 贝尔不等式的数学形式：|E(a,b) - E(a,b')| + |E(a',b) + E(a',b')| ≤ 2
- CHSH 不等式形式：|S| ≤ 2，其中 S = E(a,b) - E(a,b') + E(a',b) + E(a',b')
- 关键参数：光子偏振角度 a=0°, a'=22.5°, b=45°, b'=67.5°
- 预期结果：|S| ≈ 2.7（违反经典极限，支持量子力学）

## 中间结论
- 实验需要：纠缠光子对、快速偏振测量、随机设置选择
-  loopholes：局域性漏洞（距离不够）、探测效率漏洞（低于90%）

## 待解决
- 如何确保测量设置在光子飞行过程中完成（消除局域性漏洞）
```

## L2 工作记忆（Working）

**内容：** 当前子角色任务链的完整上下文
**生命周期：** 归入 vm/log/archive/
**存储位置：** vm/memory/context/L2-session.md
**用途：** 记住本次子角色会话的完整上下文链

---

## L3 知识记忆（Knowledge）

**内容：** 经过验证的子角色专业知识
**生命周期：** 归入 sections/ 或 sop/
**存储位置：** 不在此处长期存，验证后归 sections/
**用途：** 验证过的知识 → 写进 sections，不在此重复存储

---

## L4 元记忆（Meta）

**内容：** 关于"这个子角色擅长什么、不擅长什么"的自我认知
**生命周期：** 永久（随角色存在）
**存储位置：** sections/01-identity.md 的能力矩阵区域
**用途：** 自我认知 — 知道自己的边界和成长方向

---

## 记忆流转规则

```
L0 → 对话结束 → 丢弃（或提炼有价值内容 → 写L1）
L1 → 任务结束 → 整理后归档到 L2
L2 → 角色切换回总灵魂 → 归档到 vm/log/archive/
L3 → 验证通过 → 写入 sections/ 或 sop/，从此不在 memory 里存放
L4 → 每次角色使用后更新（能力矩阵变化时）
```
