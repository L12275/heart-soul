# Git 合流（Git Tree）

> 记录 soul-doc-system 的多分支合流历史和各模型分支的内容流向。
> 这一个合流仓库中，各模型各自维护自己的隔离区，只在主分支 merge 精华。

## 目录结构

| 目录 | 用途 |
|------|------|
| convergence/ | 合流记录（哪些分支、合流了什么、主题、时间戳） |
| history/ | git 提交历史摘要 |
| remotes/ | 远程仓库地址 |

## 远程仓库

- GitHub: https://github.com/L12275/heart-soul
- Gitee: https://gitee.com/L12275/heart-soul

## 合流记录格式

\```markdown
# [合流编号] — [合并内容简述]
**时间：** YYYY-MM-DD HH:MM
**分支：** [分支名称]
**内容：** [合并了什么到主分支]
**参与模型：** [哪些模型参与了内容生产]
\```
