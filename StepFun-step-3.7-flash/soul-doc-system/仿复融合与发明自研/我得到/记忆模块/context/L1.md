## 格式

```
[高频场景] → [直接指向L3文件名]
[低频场景] → [关键词]，read L2 / ls L3 自行定位
```

## 8条规则（GA设计）

1. 高频场景直接映射到L3文件名（快速直达）
2. 低频场景只列关键词，自行在L2/L3查找
3. 不超过30行
4. 只写"有什么能力"和"对应文件路径"
5. 不写细节，细节在L3（SOP）里
6. 两级映射：高频→直接路径，低频→关键词
7. 每次对话开始时加载
8. 每次工作结束后更新
