# 过河桥 — 元规则/合体/

> 合体 = 归档的组合版SOP。
> 当多个SOP合并为新的统一SOP后，旧版本移入此文件夹。

## 用户原话重述
合体文件夹存放已归档的组合版SOP。当新的组合版SOP创建后，旧版本移入此文件夹。

## 理解要点
- 合体 = 归档区，不是活跃区
- 新组合版 → 实时加载
- 旧版本 → 合体
- 合体内的文件 = 历史记录，不删除
