# CLI 化工具 — 示例脚本

> 用 CLI-Anything 框架封装的命令行工具示例。
> 模型可随时按此模式封装新的 CLI 工具。

## 用法

```bash
python <工具名>.py [参数]
```

## 工具列表

| 脚本 | 功能 | 用法 |
|------|------|------|
| browser-screenshot.py | 网页截图 | `python browser-screenshot.py <url> <output.png>` |
| fetch.py | HTTP GET/POST | `python fetch.py <url> [--method POST] [--data '{}']` |
