# 代码工作台 — Wiki

> 知识、事实、参考文档的永久仓库。

## 分类

| 类别 | 存放位置 | 说明 |
|------|---------|------|
| 参考 | wiki/references/ | 外部文档链接和笔记 |
| 历史 | wiki/history/ | 项目历史记录 |
| 知识 | wiki/knowledge/ | 编码知识、模式、技巧 |
| 事实 | wiki/facts/ | 已验证的事实和结论 |

## 规则

- Wiki内容 = 经过验证的、长期有效的知识
- 不存放临时内容（临时内容放 collection/）
- 每条知识必须有来源标注
- 过期内容标记为 [已过时] 而非删除
