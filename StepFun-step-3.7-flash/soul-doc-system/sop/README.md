# 通用SOP

> 可复用标准操作流程。
> **当本文件夹下文件超过 20 个时，必须自觉按功能域分类到子文件夹中（如 超级类/、信息类/、因果类/、团队类/）。不要堆积在一个目录里。**
> **所有SOP必须结合 `超级信息SOP.md` 完成，避免潜在问题，成品更少错。**

---

## 可用SOP

- `超级信息SOP.md` — 信息输出铁律 + 数据因果链 + 信息闭环
- sop/ 目录下的所有SOP文件

## SOP格式

```
SOP名称
触发条件：什么情况下使用
步骤：...
验证方式：怎么知道做对了
```

## 归纳规则

同一场景出现3次 => 自动提炼为SOP候选 => 验证1次 => 标记validated
