# 十四、工作台（Workbench）

> 模型的"工作台面"——像木匠、石匠、画家的台面。
> 
> **不是办公桌。工作台 ≠ 工作流 ≠ 通讯录 ≠ 收藏夹。**
>
> 工作台是一个区域的多种可能性的快速组合。
> 一套与画有关的注意力工具放在一个台面上——一看就知道很多种用法、解法、机会、预期。
> 它高于 SOP 和工作流——不是固定流程，是能在一个台面里搭配出所有种工作用途和工作流。

**核心区别：**
- 工作流：固定流程，按步骤执行
- 工作台：多种工具组合在一个空间，随时切换不同用法
- 类比：工作流是菜谱（精确步骤），工作台是厨房（有锅有灶有食材，能做多道菜）

---

## 14.2 标准工作台类型

| 工作台 | 用途 | 必备内容 |
|--------|------|---------|
| 音乐工作台 | 音频相关任务 | DAW路径、音源库、导出配置 |
| 视频工作台 | 视频相关任务 | 剪辑软件路径、素材库、导出模板 |
| 文章工作台 | 写作相关任务 | 编辑器路径、参考文献库、排版模板 |
| 研究工作台 | 研究相关任务 | 资料来源、分析工具、笔记模板 |
| 代码工作台 | 编程相关任务 | IDE路径、项目结构、运行环境 |
| 项目工作台 | 项目管理 | 任务跟踪、依赖图、交付物模板 |
| 科学工作台 | 科学计算 | 计算环境、数据集、仿真配置 |

## 14.3 工作台 README 模板

```markdown
# [类型]工作台

> 为此工作台配齐全部所需内容和路径

## 配置
- 工具路径：[绝对路径]
- 配置文件：[绝对路径]
- 接口：[API endpoint / 端口]

## 上次工作状态
- 时间：YYYY-MM-DD
- 状态：进行中/已暂停/已完成
- 下一步：[具体步骤]
- 待解决问题：[如有]

## 文件树
关键文件和路径（平面坐标导航图——一篇文档快速直达）
```
