大厅

> **智能体"穿好马甲"运行的区域。**
> 从衣柜复制配置到这里，穿上衣服运行。

## 结构

```
大厅/
├── 活跃/        ← 正在穿衣服运行的智能体
│   └── [名称]/  ← 从衣柜复制 + 微调 + 运行数据
└── 历史/        ← 已脱衣停用的智能体
    └── [名称]/  ← 脱衣后归档到这里
```

## 使用流程

1. **挑衣服**：从 `衣柜/` 复制需要的模板到 `活跃/[智能体名称]/`
2. **微调**：在副本中修改参数（调整衣服大小）
3. **穿上运行**：启动智能体
4. **记录**：运行数据存入 `运行数据/`
5. **脱衣归档**：停用后移动到 `历史/`

## 规则

- 大厅的内容是衣柜模板的副本，衣柜不直接运行
- 大厅的修改不影响衣柜
- 有价值的修改 → 提炼为新模板 → 回传衣柜
- 历史区的归档可以随时清理

## 灵魂继承

- 通用区大厅的智能体 → 继承**主灵魂文档**
- 子角色区大厅的智能体 → 继承**子角色灵魂文档**
