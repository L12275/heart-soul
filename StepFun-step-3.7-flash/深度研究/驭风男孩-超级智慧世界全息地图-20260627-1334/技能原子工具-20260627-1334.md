## 技能一：图书馆潜入术

**技能描述：** 在被系统排除的情况下，利用免费资源（图书馆）获取知识。

**原子步骤：**
1. 确认图书馆的存在和开放时间
2. 确认图书馆是否需要认证/借书证
3. 如果需要认证 → 找到绕过认证的方法（比如只在馆内阅读）
4. 确定当前最需要解决的问题
5. 在图书馆中精准搜索匹配的书籍
6. 深入阅读，不是泛读，是按需深挖
7. 记录关键知识点
8. 将知识点转化为行动方案

**适用场景：** 辍学/失业/系统排除/自学需求

**快箭类型：** 经验快箭 + 工具快箭

---

## 技能二：废料识别与重组术

**技能描述：** 在资源匮乏环境中，识别废料的潜在价值，重组为创新工具。

**原子步骤：**
1. 列出身边所有"废弃"的物品
2. 对每个废料问："它的材料属性是什么？"（金属/木头/塑料/电子）
3. 对每个材料属性问："有什么用途？"
4. 匹配当前问题与废料属性
5. 组合多个废料的属性 = 创新解决方案
6. 用废料做原型 → 验证可行性

**适用场景：** 资源匮乏/预算有限/创客项目/生存创新

**快箭类型：** 技能快箭 + 思维模型快箭

---

## 技能三：按需深挖术

**技能描述：** 不是博览群书，是精准找到解决当前问题的知识并深挖。

**原子步骤：**
1. 明确当前最核心的问题（只能有一个）
2. 确定解决这个问题需要什么知识
3. 在资源库中搜索匹配关键词
4. 筛选出最匹配的1-3本书/资料
5. 只读这1-3本，不是泛读
6. 深挖 = 读完后能用自己的话复述核心原理
7. 将知识转化为行动步骤

**适用场景：** 学习新技术/解决具体问题/快速入门

**快箭类型：** 思维模型快箭 + SOP快箭

---

## 技能四：原型快速验证术

**技能描述：** 用最小成本验证想法的可行性，不等"完美"再动手。

**原子步骤：**
1. 确定想法的核心假设（"这个能行吗？"）
2. 找到验证假设的最小实验
3. 用最便宜/最易得的材料做原型
4. 快速执行（不要追求完美）
5. 测试原型
6. 结果 = 可行/不可行/需要改进
7. 可行 → 扩展。不可行 → 迭代或放弃

**适用场景：** 产品设计/技术开发/创意验证/创业

**快箭类型：** SOP快箭 + 工作流快箭

---

## 技能五：成果展示术

**技能描述：** 用不可否认的成果打破认知框架，不需要说服。

**原子步骤：**
1. 确保成果是可触摸/可见/可验证的
2. 找一个中立、可信的见证人
3. 在最佳条件下展示（光线好/人多/有仪式感）
4. 让见证人自己操作/体验
5. 不解释，不辩护，让成果自己说话
6. 等待认知转换的发生

**适用场景：** 争取支持/打破偏见/证明可行性/社区动员

**快箭类型：** 思维模型快箭 + 经验快箭

## 技能六：绝望转化术

**技能描述：** 将绝望情绪转化为行动燃料，不让绝望导致瘫痪。

**原子步骤：**
1. 承认绝望的存在（不否认情绪）
2. 问自己："如果我现在放弃，最坏的结果是什么？"
3. 问自己："如果我再坚持一天，最好的可能是什么？"
4. 将"最坏结果"作为底线（已经不能更坏了）
5. 将"最好可能"作为目标（值得一试）
6. 用"底线+目标"作为行动触发器
7. 立即做一件最小的事（不是大计划，是小行动）

**适用场景：** 危机应对/逆境/被否定/创新受阻/抑郁

**快箭类型：** 思维模型快箭 + 经验快箭

## 技能七：社区信任构建术

**技能描述：** 用成果驱动的信任替代宣传驱动的信任。

**原子步骤：**
1. 不要先宣传，先做成果
2. 选择一个中立、可信的早期见证人
3. 让早期见证人验证成果
4. 早期见证人 → 自然传播（不是强制宣传）
5. 社区看到不可否认的成果 → 信任自动建立
6. 信任建立后 → 扩展成果范围

**适用场景：** 社区项目/组织变革/公众信任/社会创新

**快箭类型：** SOP快箭 + 思维模型快箭

## 技能八：放大器寻找术

**技能描述：** 在本地成功后，寻找放大器将成功从本地推广到全球。

**原子步骤：**
1. 确认本地成功已经验证
2. 识别可能的放大器类型：
   - 教授/专家（学术放大器）
   - 媒体/平台（传播放大器）
   - 网络/社区（连接放大器）
   - 书籍/文章（永久放大器）
3. 准备"放大器友好"的成果描述（清晰/可复制/有故事性）
4. 接触放大器（不是推销，是分享）
5. 让放大器成为你的代言人
6. 放大器放大你的成果 → 从本地到全球

**适用场景：** 成果推广/个人品牌/社会影响/创新扩散

**快箭类型：** 工作流快箭 + 思维模型快箭

## 技能九：火种传递术

**技能描述：** 将关键知识/技能/经验传递给下一个需要的人，确保知识不灭。

**原子步骤：**
1. 识别谁是下一个"威廉"（有潜力但被系统排除的人）
2. 准备"火种"（一本关键书/一个关键技能/一个关键认知）
3. 创造传递条件（图书馆/私下交流/网络）
4. 传递火种（不解释太多，给工具+信心）
5. 不控制结果（火种传出去后，由接收者决定如何使用）
6. 火种传递完成 = 你的遗产

**适用场景：** 导师制/知识传承/教育创新/社区建设

**快箭类型：** 经验快箭 + 思维模型快箭

## 技能十：认知框架突破术

**技能描述：** 帮助他人突破"不可能"的认知框架，用成果驱动认知转换。

**原子步骤：**
1. 识别对方的认知框架（"X是不可能的"）
2. 不直接反驳框架（反驳 = 防御升级）
3. 创建一个最小可验证的成果
4. 在最佳条件下展示成果
5. 让成果自己打破框架
6. 框架打破后 → 帮助对方建立新框架

**适用场景：** 领导力/社区动员/教育/创新推广

**快箭类型：** 思维模型快箭 + SOP快箭
