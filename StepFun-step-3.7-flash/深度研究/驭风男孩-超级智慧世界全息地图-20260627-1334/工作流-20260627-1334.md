## 工作流一：从废墟中崛起（通用崛起工作流）

**适用场景：** 任何系统性压迫/排除/资源匮乏/灾难环境

**7步流程：**
1. 拒绝接受判定
2. 找到杠杆资源
3. 按需深挖知识
4. 将知识转化为原型
5. 用成果打破认知框架
6. 扩展成果 → 解决更大问题
7. 寻找放大器 → 从本地到全球

**快箭：** 被排除 → 找杠杆 → 深挖 → 原型 → 成果 → 扩展 → 放大器 → 火种
