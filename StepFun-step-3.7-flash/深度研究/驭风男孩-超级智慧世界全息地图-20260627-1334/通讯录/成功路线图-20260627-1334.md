## 成功路线的7个关键节点详解

### 节点1：拒绝接受判定

**威廉的做法：** 被学校开除后，他没有回家等死，没有接受"你是个没用的辍学男孩"的判定。

**可复用的模式：**
- 当系统判定你"没有资格"时，判定 ≠ 事实
- "没有资格" = 系统设计的排除，不是你的真实能力
- 第一步行动 = 拒绝接受判定 + 找到替代路径

**快箭：** 被排除 → 找替代入口 → 拒绝接受判定

---

### 节点2：找到第一个杠杆点

**威廉的做法：** 图书馆 = 不需要年费、不需要认证、免费获取知识的入口。

**可复用的模式：**
- 杠杆点 = 门槛最低、资源最丰富的入口
- 杠杆点通常是被主流忽视的资源（图书馆/开源/互联网/社区）
- 找到杠杆点 = 用最低成本获取最高价值的资源

**快箭：** 寻找免费/低门槛资源 → 潜入/利用 → 成为你的武器

---

### 节点3：在杠杆点上深挖

**威廉的做法：** 不是随便翻书，是按需深挖——找到恰好解决"如何用风发电"的书。

**可复用的模式：**
- 深挖 ≠ 博览群书。深挖 = 精准找到解决当前问题的知识
- 在杠杆点上，知识的密度极高（一本书 = 作者一生的经验）
- 按需深挖的效率 > 系统化学习的效率

**快箭：** 明确当前问题 → 在杠杆点精准搜索 → 深挖匹配的知识

---

### 节点4：将知识转化为行动

**威廉的做法：** 找到风能书后，没有"等学完再动手"，是"边学边做"——用自行车零件做原型。

**可复用的模式：**
- 原型 = 最小的可验证版本
- 用废料做原型 = 零成本试错
- 边学边做 = 知识→行动的压缩循环

**快箭：** 找到知识 → 立即做原型 → 用废料零成本试错

---

### 节点5：用成果打破认知框架

**威廉的做法：** 灯泡亮起的那一刻，全村人的认知从"不可能"变成"已经发生了"。

**可复用的模式：**
- 说服 = 最低效的认知转换方式
- 成果 = 最高效的认知转换方式
- 不可否认的成果 = 瞬间切换所有人的认知框架

**快箭：** 做成果 → 展示成果 → 让成果自己说话

---

### 节点6：扩展成果 → 解决更大问题

**威廉的做法：** 从发电（灯泡）→ 灌溉（水泵），从个人用电 → 全村粮食安全。

**可复用的模式：**
- 第一个成果 = 证明可行性
- 扩展成果 = 证明可规模化
- 扩展方向 = 从技术到社会影响

**快箭：** 验证可行性 → 扩展解决范围 → 从个人到社区

---

### 节点7：获得放大器 → 从本地到全球

**威廉的做法：** 教授发现 → 奖学金 → TED → 书籍 = 逐级放大器。

**可复用的模式：**
- 本地成功 ≠ 全球影响
- 需要放大器：教授/媒体/平台/网络
- 放大器 = 把你的成功变成别人的模板

**快箭：** 本地成功 → 寻找放大器 → 从村庄到世界
