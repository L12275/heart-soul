## 关键关系链与成败因果

### 链一：威廉 → 父亲 → 风车 → 全村

| 节点 | 关系 | 因果 |
|------|------|------|
| 威廉 | 提出造风车 | 父亲认为"疯了" |
| 父亲 | 从怀疑到支持 | 看到灯泡亮起 → 态度180度转变 |
| 风车 | 从废料到电力 | 自行车零件 → 发电 → 全村用电 |
| 全村 | 从嘲笑到依赖 | 灯泡亮起 → 认知转换 → 信任 |
| **因果结论** | **一个人的坚持 → 改变父亲 → 改变社区** |

**成败关键：** 威廉没有放弃。如果他放弃了，父亲永远不会看到灯泡，全村永远不会知道风车能发电。

---

### 链二：迈克 → 威廉 → 图书馆 → 风车

| 节点 | 关系 | 因果 |
|------|------|------|
| 迈克 | 送直流发电机教材 | 火种传递 |
| 威廉 | 潜入图书馆 | 找到风能书 |
| 图书馆 | 知识的物理载体 | 免费资源 = 被排除者的工具 |
| 风车 | 从书到实物 | 知识 → 行动 → 成果 |
| **因果结论** | **一个老师的礼物 → 改变一个学生 → 改变一个村庄** |

**成败关键：** 迈克的礼物（直流发电机教材）是威廉的起点。如果迈克没有送这本书，威廉可能永远不会找到风能书。

---

### 链三：政府 → 断电/欠薪/学费 → 威廉辍学 → 自学 → 风车

| 节点 | 关系 | 因果 |
|------|------|------|
| 政府 | 断电+欠薪+学费门槛 | 系统性压迫 |
| 威廉 | 被迫辍学 | 被系统排除 |
| 自学 | 潜入图书馆 | 系统外的学习 |
| 风车 | 用自学知识造风车 | 排除反而催生创新 |
| **因果结论** | **系统的压迫 → 个体的反抗 → 超越系统** |

**成败关键：** 系统的压迫（断电/欠薪/学费）反而把威廉推到了自学道路上。如果威廉没有被开除，他可能永远不会深入图书馆。

---

### 链四：防洪林 → 砍林 → 洪水 → 饥荒 → 风车灌溉

| 节点 | 关系 | 因果 |
|------|------|------|
| 防洪林 | 保护村庄 | 正常状态 |
| 村民 | 砍林卖钱 | 短期生存决策 |
| 洪水 | 冲毁庄稼 | 长期灾难 |
| 饥荒 | 全村挨饿 | 系统性崩溃 |
| 风车 | 灌溉 + 电力 | 从饥荒中拯救村庄 |
| **因果结论** | **短期决策的灾难 → 长期创新的机遇** |

**成败关键：** 砍林导致饥荒，饥荒催生风车。如果没有饥荒，威廉可能永远不会急着造风车。

---

### 链五：姐姐私奔 → 家庭粮食压力 → 威廉的责任感 → 更快造风车

| 节点 | 关系 | 因果 |
|------|------|------|
| 安妮 | 私奔减轻粮食压力 | 系统性代价 |
| 家庭 | 少了一口人吃饭 | 短暂缓解 |
| 威廉 | 看到姐姐离开 + 父亲绝望 | 责任感加剧 |
| 风车 | 更快建造 | 责任感 = 行动加速器 |
| **因果结论** | **个人代价 → 催化更大行动** |

**成败关键：** 安妮的离开让威廉更坚定地要造风车。如果安妮没有走，威廉可能不会那么急。

## 成功条件清单（从关系网络推导）

1. **有一个坚持的人** — 威廉的 stubborn 是不可替代的
2. **有一个传递火种的人** — 迈克的教材是起点
3. **有免费的资源** — 图书馆 = 被排除者的武器
4. **有一个可见的成果** — 灯泡亮起 = 认知转换的触发器
5. **有一个放大器** — 马尔科姆教授 + TED = 从村庄到世界
6. **社区愿意接受** — 村民从嘲笑到信任 = 成果驱动的转换
7. **家庭不彻底崩塌** — 父母没有完全放弃 = 威廉有后方

**缺一不可。** 任何一个条件缺失，故事都会有不同的结局。
