## 一、风能基础原理

### 风能的物理原理

**核心公式：** P = 0.5 × ρ × A × v³

- P = 功率（瓦特）
- ρ = 空气密度（约1.225 kg/m³ 海平面）
- A = 叶片扫掠面积（m²）
- v = 风速（m/s）

**关键发现：** 功率与风速的**三次方**成正比。风速翻倍 = 功率8倍。这就是为什么风车需要高塔架——在地面以上风速更高。

### 风车的基本结构

| 部件 | 功能 | 威廉使用的材料 |
|------|------|-------------|
| 塔架 | 支撑风车，升高到更高风速区 | 旧自行车车架 |
| 叶片 | 捕获风能，转换为旋转机械能 | 旧自行车轮 + 本地木材 |
| 轮毂 | 连接叶片和传动轴 | 废金属 |
| 传动轴 | 传递旋转机械能 | 自行车零件 |
| 发电机 | 将机械能转换为电能 | 直流发电机（迈克老师赠送） |
| 底座 | 固定整个结构 | 本地废料/混凝土 |

## 三、风能技术的可复用知识

### 小型风力发电系统的关键设计原则

1. **塔架高度 > 叶片大小** — 高塔架 = 更高风速 = 更多发电
2. **叶片角度 = 效率关键** — 45度是常见最优角度
3. **发电机匹配** — 发电机功率必须匹配风车输出功率
4. **废料可用性** — 在资源匮乏环境中，废料 = 主要材料来源
5. **零成本试错** — 用废料做原型 = 失败了也不损失

### 不同环境的风能适用性

| 环境 | 风速 | 适用性 | 建议 |
|------|------|--------|------|
| 非洲草原 | 中高（干季） | 高 | 适合小型风车 |
| 山区 | 高 | 高 | 适合小型到中型风车 |
| 城市 | 低（建筑物遮挡） | 低 | 不适合，用太阳能 |
| 海岸 | 高 | 高 | 适合中型到大型风车 |
| 平原 | 中 | 中 | 适合小型风车 |

## 四、风能 vs 其他能源的对比

| 能源类型 | 成本 | 适用环境 | 威廉环境的选择 |
|---------|------|---------|--------------|
| 风能 | 极低（废料） | 风大地区 | ✅ 选择了风能 |
| 太阳能 | 中等 | 日照充足地区 | 马拉维日照也可，但威廉没找到太阳能书 |
| 水力 | 高 | 有河流地区 | 马拉维有河流但需要基础设施 |
| 化石燃料 | 高 | 任何地方 | 马拉维买不起 |
| 电网 | 系统提供 | 有电网地区 | 马拉维政府断电 |

## 五、威廉风车的技术遗产

### 威廉之后的风车技术发展

1. **更大规模的风车** — 从4盏灯到全村用电
2. **风车+太阳能混合** — 非洲发明家网络的常见方案
3. **风车灌溉系统** — 从发电到农业的扩展
4. **低成本风车设计** — 用废料/本地材料 = 可复制的技术

### 风车技术的全球影响

- 威廉的故事 = 风能在发展中国家应用的全球模板
- 风车 = 分布式能源的象征
- 低成本风车 = 贫困地区能源独立的路径
