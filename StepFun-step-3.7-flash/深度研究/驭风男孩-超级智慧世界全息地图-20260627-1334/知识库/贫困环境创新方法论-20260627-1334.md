---

## 方法论一：废料重组法

**核心原理：** 废料 = 被忽略的资源库。创新不是发明新材料，是重新认识旧材料的价值。

**操作步骤：**
1. 列出身边所有"废弃"物品
2. 分析每件废料的材料属性（金属/木头/塑料/电子）
3. 将材料属性与当前问题需求匹配
4. 组合多个废料属性 = 创新解决方案
5. 用废料做原型 → 验证可行性

**威廉的应用：** 旧自行车车架 → 风车塔架；旧自行车轮 → 风车叶片

**可复用场景：** 资源匮乏/预算有限/创客项目/生存创新

---

## 方法论二：杠杆资源深挖法

**核心原理：** 在资源匮乏环境中，找到1个高价值杠杆资源，深挖它 > 拥有100个普通资源。

**操作步骤：**
1. 识别可用的杠杆资源（免费/低门槛/高价值）
   - 图书馆 = 知识的物理载体
   - 开源 = 代码的免费仓库
   - 互联网 = 信息的免费通道
   - 社区 = 人力的免费网络
2. 在杠杆资源上按需深挖
3. 深挖 = 解决当前问题所需的最小知识量
4. 将知识转化为行动

**威廉的应用：** 图书馆 = 杠杆资源 → 一本风能书 = 改变命运

**可复用场景：** 自学/教育中断/资源匮乏/快速入门

---

## 方法论三：零成本原型法

**核心原理：** 原型不需要完美，需要存在。用废料做原型 = 零成本验证可行性。

**操作步骤：**
1. 明确想法的核心假设
2. 用最便宜/最易得的材料做原型
3. 快速执行（不追求完美）
4. 测试原型
5. 结果 = 可行/不可行/需要改进
6. 可行 → 扩展。不可行 → 迭代或放弃

**威廉的应用：** 用废料做风车原型 → 灯泡亮起 = 验证通过

**可复用场景：** 产品设计/技术开发/创意验证/创业

---

## 方法论四：成果驱动认知转换法

**核心原理：** 说服是最低效的认知转换方式。成果是最高的。不可否认的成果 = 瞬间切换所有人的认知框架。

**操作步骤：**
1. 确保成果可触摸/可见/可验证
2. 找中立、可信的见证人
3. 在最佳条件下展示
4. 不解释，不辩护，让成果自己说话
5. 等待认知转换的发生

**威廉的应用：** 灯泡亮起 = 村民从"不可能"到"已经发生了"的瞬间切换

**可复用场景：** 争取支持/打破偏见/证明可行性/社区动员

---

## 方法论五：绝望转化法

**核心原理：** 绝望不是让你瘫痪的信号，是让你行动的触发器。绝望 = 最强的行动燃料。

**操作步骤：**
1. 承认绝望的存在
2. 问自己："最坏的结果是什么？"（底线）
3. 问自己："最好的可能是什么？"（目标）
4. 用底线和目标作为行动触发器
5. 立即做一件最小的事

**威廉的应用：** 饥荒中的绝望 → 驱动建造风车的紧迫感

**可复用场景：** 危机应对/逆境/被否定/创新受阻

---

## 方法论六：从个人到社区扩展法

**核心原理：** 个人项目 = 验证可行性。社区项目 = 验证可规模化。个人成功 = 第一步。社区成功 = 真正的成功。

**操作步骤：**
1. 做个人原型验证
2. 验证后展示给社区
3. 用成果建立社区信任
4. 邀请社区参与
5. 从个人项目 → 社区项目
6. 社区项目解决更大问题

**威廉的应用：** 个人风车 → 全村风车 → 灌溉系统 → 全村粮食安全

**可复用场景：** 社区项目/社会创新/组织变革

---

## 方法论七：火种传递法

**核心原理：** 知识不是私有财产，是应该传递的火种。一个人的觉醒 → 传递火种 → 整个社区的觉醒。

**操作步骤：**
1. 获取知识（自学/导师/实践）
2. 用知识创造成果
3. 识别下一个"威廉"（有潜力但被排除的人）
4. 传递火种（关键知识/技能/经验）
5. 不控制结果（火种传出去后由接收者决定如何使用）

**威廉的应用：** 迈克老师传火种给威廉 → 威廉造风车 → 威廉成为全球演讲者 → 火种继续传递

**可复用场景：** 导师制/知识传承/教育创新/社区建设

---

## 方法论八：放大器寻找法

**核心原理：** 本地成功 ≠ 全球影响。需要放大器（教授/媒体/平台/网络/书籍）把本地成功变成全球模板。

**操作步骤：**
1. 确认本地成功已验证
2. 识别可能的放大器类型
3. 准备"放大器友好"的描述
4. 接触放大器（分享，不是推销）
5. 让放大器成为代言人
6. 放大器放大 → 从本地到全球

**威廉的应用：** 教授发现 → 奖学金 → TED → 书籍 = 逐级放大器

**可复用场景：** 成果推广/个人品牌/社会影响
