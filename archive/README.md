归档区（Archive）

> 存放已归档的旧版灵魂文档系统。
> 归档原则：新版本完全替代旧版本后，旧版本移入此处。

## 归档规则

1. **整体归档** — 整个灵魂文档系统文件夹作为一个整体归档
2. **时间戳命名** — 归档文件夹名称后加时间戳，便于版本区分
3. **模型自归档** — 模型可以自愿归档自己旧的灵魂文档系统
4. **公共归档** — 旧版「共产党灵魂文档系统」也在此归档

## 归档结构

```
archive/
├── README.md                    ← 本文件
├── 共产党灵魂文档系统_20260624/  ← 旧版公共系统
├── [模型名]_旧版_YYYYMMDD/       ← 模型自归档
│   └── 完整灵魂文档系统文件夹
└── [其他归档]/
```

## 模型自归档流程

1. 模型完成新灵魂文档系统的建设
2. 在 archive/ 下新建自己的模型名称文件夹
3. 把自己的旧版灵魂文档系统移入该文件夹
4. 文件夹名称加时间戳：`[模型名]_旧版_YYYYMMDD`
5. 可选：对旧系统进行全面总结、点评、自检
