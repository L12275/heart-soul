# 技能库（Skills）

> 这个子角色的专业技能工具包。
> 每个技能是一个可复用的能力模块。

## 技能清单

| 技能名称 | 类型 | 状态 | 熟练度 |
|---------|------|------|--------|
| （暂无） | — | 待积累 | — |

## 如何添加技能

1. 在此目录下创建 [技能名称].md
2. 按照以下格式编写：

```markdown
# [技能名称]

**触发条件：** 什么情况下使用此技能
**执行步骤：**
1. [步骤1]
2. [步骤2]
3. [步骤3]
**检查方式：** 如何验证技能执行正确
**熟练度：** [初始/熟练/精通]
```

## 技能熟练度

- **初始**：能用但需对照步骤
- **熟练**：能流畅使用
- **精通**：能根据场景灵活变通

## 技能来源

- 可以从总灵魂的技能库借鉴
- 可以从外部学习新技能
- 必须在 genome/ 中标注技能来源
