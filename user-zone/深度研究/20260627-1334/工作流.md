## 工作流一：从废墟中崛起（通用崛起工作流）

**适用场景：** 任何系统性压迫/排除/资源匮乏/灾难环境

**7步流程：**
1. 拒绝接受判定
2. 找到杠杆资源
3. 按需深挖知识
4. 将知识转化为原型
5. 用成果打破认知框架
6. 扩展成果 → 解决更大问题
7. 寻找放大器 → 从本地到全球

**快箭：** 被排除 → 找杠杆 → 深挖 → 原型 → 成果 → 扩展 → 放大器 → 火种

---

## 工作流二：风车建造（技术工作流）

**适用场景：** 任何需要从零开始构建技术解决方案

**8阶段流程：**
1. 问题识别
2. 资源扫描
3. 知识获取
4. 设计原型
5. 材料收集
6. 原型建造
7. 验证与扩展

**快箭：** 问题 → 资源 → 知识 → 设计 → 收集 → 建造 → 验证 → 扩展

---

## 工作流三：社区拯救（社会创新工作流）

**适用场景：** 社区危机/社会问题/集体行动

**6步流程：**
1. 识别危机
2. 个人行动（不等别人）
3. 成果展示
4. 社区信任建立
5. 集体行动
6. 系统改变

**快箭：** 危机 → 个人行动 → 成果 → 信任 → 集体 → 系统改变
