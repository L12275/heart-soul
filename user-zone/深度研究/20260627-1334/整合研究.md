## 维度一：灵魂维度（为什么）

### 核心问题

**为什么一个13岁辍学男孩能改变整个村庄的命运？**

### 灵魂答案

不是因为威廉聪明，是因为：
1. **他拒绝了系统的判定** — "没有资格" ≠ "没有能力"
2. **他找到了杠杆资源** — 图书馆 = 被排除者的武器
3. **他用废料做创新** — 旧自行车 = 风车的核心
4. **他不等待救世主** — 自己就是救世主
5. **他为社区而做** — 不是为了自己用电，是为了全村不饿死

### 灵魂公式

```
被系统排除 → 找杠杆资源 → 废料创新 → 成果驱动 → 社区拯救 → 全球影响
```

## 维度二：社会维度（谁/关系）

### 核心问题

**人物关系如何决定成败？**

### 社会答案

- **威廉** = 觉醒者（一个人看见别人看不见的）
- **父亲** = 从怀疑到支持（态度转变 = 成果驱动）
- **迈克老师** = 火种传递者（知识的桥梁）
- **村民** = 从嘲笑到信任（认知转换 = 成果驱动）
- **教授** = 放大器（从村庄到世界）
- **政府** = 压迫者（也是机遇的制造者）

### 关系公式

```
一个人的觉醒 → 父亲的态度转变 → 社区信任建立 → 教授放大 → 全球影响
```

### 成功条件

1. 有一个坚持的人（威廉）
2. 有一个传递火种的人（迈克）
3. 有免费资源（图书馆）
4. 有可见成果（灯泡亮起）
5. 有放大器（教授/TED）
6. 社区愿意接受（从嘲笑到信任）
7. 家庭不彻底崩塌（父母没有完全放弃）

**缺一不可。**

## 维度三：技术维度（怎么做）

### 核心问题

**风车是怎么造出来的？**

### 技术答案

```
问题识别（缺电+缺水+饥荒）
  → 资源扫描（旧自行车+废料+图书馆）
  → 知识获取（《用风能发电》）
  → 设计原型（塔架+叶片+发电机）
  → 材料收集（拆自行车+收废料）
  → 建造调试（装起来→调角度→等风）
  → 验证（灯泡亮起）
  → 扩展（灌溉+全村用电）
```

### 关键参数

- 塔架高度：约5米
- 叶片：4片（自行车轮改造）
- 发电机：直流发电机（迈克赠送）
- 发电量：4盏灯 + 收音机
- 灌溉：全村农田
