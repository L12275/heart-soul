# 驭风男孩 — 可行行动方案

> **文件性质：从超级智慧世界中提取的可执行行动方案。**
> 本文不是"分析电影"，是"把电影的智慧转化为你可以今天就开始的行动"。
> 每个行动方案 = 一条快箭，下次遇到类似场景时直接调用。

---

## 行动方案一：被系统排除时的崛起路线

### 适用场景

- 被学校开除/退学
- 被公司开除/失业
- 被社区排斥
- 系统判定你"没有资格"
- 没有学历/证书/认证

### 7步行动

| 步骤 | 行动 | 今日可做 |
|------|------|---------|
| 1. 拒绝判定 | 不接受"没有资格"的判定 | 写下3个你被排除的理由，然后写3个反例 |
| 2. 找杠杆资源 | 找免费/低门槛资源 | 今天找一个免费学习资源（图书馆/开源/互联网） |
| 3. 按需深挖 | 精准解决当前问题 | 明确你最核心的1个问题，搜索匹配的知识 |
| 4. 原型验证 | 用废料做最小原型 | 今天用身边的东西做一个最小原型 |
| 5. 成果展示 | 做可验证的成果 | 做一件不可否认的小成果，展示给1个人 |
| 6. 社区扩展 | 从个人到社区 | 把你的成果分享给社区/朋友圈 |
| 7. 寻找放大器 | 从本地到更大范围 | 准备一个清晰的故事描述 |

### 快箭

```
被排除 → 找杠杆 → 深挖 → 原型 → 成果 → 扩展 → 放大器
```

## 行动方案二：绝望中的行动触发器

### 适用场景

- 处于绝境（失业/贫困/疾病/被否定）
- 觉得"不可能"
- 想放弃
- 陷入抑郁/瘫痪

### 5步行动

| 步骤 | 行动 | 今日可做 |
|------|------|---------|
| 1. 承认绝望 | 不否认情绪 | 写下你现在最绝望的事 |
| 2. 确定底线 | 最坏的结果 | 写下最坏结果（已经不能更坏了） |
| 3. 确定目标 | 最好的可能 | 写下最好可能（值得一试） |
| 4. 转化燃料 | 用底线+目标驱动 | 把底线和目标贴在你每天能看到的地方 |
| 5. 立即行动 | 做最小的事 | 今天做一件最小的事（不是大计划） |

### 快箭

```
绝望 → 底线+目标 → 行动燃料 → 立即行动
```

## 行动方案三：用废料创新

### 适用场景

- 想做创新但没钱买材料
- 资源匮乏
- 预算有限
- 创客项目

### 6步行动

| 步骤 | 行动 | 今日可做 |
|------|------|---------|
| 1. 列出废料 | 身边所有废弃物品 | 列出3件你身边的"废弃"东西 |
| 2. 分析属性 | 每件废料的材料属性 | 分析它们的材料（金属/木头/塑料/电子） |
| 3. 匹配需求 | 废料属性 vs 你的问题 | 匹配1件废料到1个当前问题 |
| 4. 做原型 | 用废料做最小原型 | 今天用废料做一个原型 |
| 5. 测试验证 | 测试原型 | 测试你的原型 |
| 6. 迭代优化 | 根据测试结果改进 | 改进你的原型 |

### 快箭

```
废料 → 分析属性 → 匹配需求 → 原型 → 验证 → 迭代
```

## 行动方案四：社区拯救路线

### 适用场景

- 社区面临危机
- 需要集体行动
- 社区不信任/分裂
- 你想改变社区

### 6步行动

| 步骤 | 行动 | 今日可做 |
|------|------|---------|
| 1. 识别危机 | 社区面临什么问题 | 写下社区最核心的1个问题 |
| 2. 个人行动 | 不等别人，自己先做 | 做一个最小可行的行动 |
| 3. 成果展示 | 做可验证的成果 | 做1个可见的成果，展示给社区 |
| 4. 建立信任 | 用成果驱动信任 | 让成果自己说话 |
| 5. 集体行动 | 邀请社区参与 | 邀请3个人参与你的行动 |
| 6. 系统改变 | 从社区到更大系统 | 准备一个更大范围的故事 |

### 快箭

```
危机 → 个人行动 → 成果 → 信任 → 集体 → 系统改变
```

## 行动方案五：知识传递火种计划

### 适用场景

- 你学到了有价值的东西
- 你想帮助被排除的人
- 你想做导师/教育者

### 5步行动

| 步骤 | 行动 | 今日可做 |
|------|------|---------|
| 1. 识别火种 | 你最核心的知识/技能 | 写下你最想传递给别人的1个知识/技能 |
| 2. 找到接收者 | 下一个"威廉" | 识别1个有潜力但被系统排除的人 |
| 3. 准备传递 | 准备关键资料 | 准备一份关键资料/技能包 |
| 4. 传递火种 | 实际传递 | 今天把火种传递给那个人 |
| 5. 不控制结果 | 火种传出去后由接收者决定 | 不要干涉接收者如何使用火种 |

### 快箭

```
获取知识 → 创造成果 → 找到接收者 → 传递火种 → 不控制结果
```

## 行动方案六：放大器寻找计划

### 适用场景

- 你的本地成功了
- 想让更多人知道
- 想扩大影响

### 5步行动

| 步骤 | 行动 | 今日可做 |
|------|------|---------|
| 1. 确认成功 | 本地成功已验证 | 确认你的成果是可触摸/可见/可验证的 |
| 2. 准备故事 | 清晰的描述 | 写一个1分钟的故事描述 |
| 3. 识别放大器 | 可能的放大器类型 | 列出3个可能的放大器（教授/媒体/平台） |
| 4. 接触放大器 | 分享不是推销 | 今天联系1个可能的放大器 |
| 5. 持续放大 | 用放大器扩大影响 | 让放大器成为你的代言人 |

### 快箭

```
本地成功 → 准备故事 → 找放大器 → 从本地到全球
```

## 行动方案七：自学成才路线

### 适用场景

- 想学新技能/新领域
- 教育中断/无法负担正规教育
- 快速入门

### 5步行动

| 步骤 | 行动 | 今日可做 |
|------|------|---------|
| 1. 明确问题 | 你最需要学什么 | 写下你最需要掌握的1个技能 |
| 2. 找杠杆资源 | 免费/低门槛资源 | 找1个高质量的学习资源 |
| 3. 按需深挖 | 只学当前需要的 | 今天学习1个核心概念 |
| 4. 做项目验证 | 用学到的知识做项目 | 做一个最小项目验证学习成果 |
| 5. 成果认证 | 用项目成果替代证书 | 把你的项目放在作品集/社交媒体 |

### 快箭

```
明确问题 → 找杠杆资源 → 按需深挖 → 做项目 → 成果认证
```

- 想放弃
- 被系统否定

### 4步行动

| 步骤 | 行动 | 今日可做 |
|------|------|---------|
| 1. 接受被排除 | 事实≠判定 | 写下系统对你的判定，然后写3个反例 |
| 2.  stubborn 宣言 | 拒绝"不可能" | 写下你的 stubborn 宣言（"我不接受不可能"） |
| 3. 找第一条路 | 最小可行路径 | 找到1条你没有被排除的路径 |
| 4. 走第一条路 | 行动 | 今天走第一步 |

### 快箭

```
被说不可能 →  stubborn 宣言 → 找路 → 走第一步
```
