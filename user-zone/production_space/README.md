# 生产空间（Production Space）

> 最大生产单位 — 包含多种工作台，可组合联动。
> 工作台可以组合成不同类型的生产空间。

---

## 结构

```
production_space/
├── README.md
└── 通用/
    ├── README.md ← 类型说明
    ├── 工作室/ ← 单一深度领域（默认推荐）
    │   ├── 音乐工作台/
    │   │   ├── README.md
    │   │   ├── desktop/sticky-notes/
    │   │   ├── workflow/
    │   │   ├── contacts/
    │   │   └── sop/ ← 专用SOP（就地取用，≠根目录通用SOP）
    │   ├── 视频工作台/
    │   ├── 文章工作台/
    │   ├── 研究工作台/
    │   ├── 代码工作台/
    │   ├── 项目工作台/
    │   └── 科学工作台/
    ├── 工厂/ ← 大规模标准化产出
    ├── 公司/ ← 综合项目管理
    ├── 作坊/ ← 小批量定制生产
    └── 创意中心/ ← 探索性/实验性产出
```

---

## 近原则

- 不同类型结构相同，只是标签不同 → 统一放在 通用/ 下
- 实际使用时默认用"工作室"，需要区分时再新建类型
- 工作台内专用内容放在工作台内，不污染根目录通用文件夹
