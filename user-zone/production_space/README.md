# 生产空间（Production Space）

> 最大的生产单位——工作室、工厂、公司、作坊、创意中心。
> 多种或多个工作台可以组合成工作室、工厂、公司等。
> 
> **纵深关系：**
> 生产空间 > 工作台 > 办公桌面 > 工作流 > 通讯录
> 
> - 生产空间：有车间走廊，多种工作台组合联动
> - 工作台：密集专注，已配齐工具，直接干活
> - 办公桌面：宽裕松散，头脑风暴、准备、实验
> - 工作流：固定流程，按步骤执行
> - 通讯录：现场高相关度联系，快速找人

**形成记忆宫殿：** 从大生产空间→到工作台→到办公桌→到工作流→到通讯录，形成了现代化生产室内到办公桌的空间由大到小到细节的垂直深度记忆宫殿。
