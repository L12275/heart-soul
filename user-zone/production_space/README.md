> 最大生产单位 — 包含多种工作台，可组合联动。

### 比喻：工厂

- **地平面级**（soul-doc-system根目录）= 地面上的公共设施
  - 工厂大门、走廊、电梯、仓库、会议室
  - 所有工作台共享的通用资源
  - 不特定于某个工作台的内容，写到这里

- **工作台级**（production_space/通用/代码工作台）= 工厂里的专用车间
  - 代码车间里面有：办公室、实验室、财务室、生产区
  - 每个车间有自己独立的桌面、工作流、通讯录、收藏夹、便签、日记、记忆
  - 只记录与这个车间高相关的内容

### 双写规则

- **通用记录**（不特定于某个工作台）→ 只写到地平面级（soul-doc-system根目录）
- **专用记录**（与某工作台高相关）→ 只写到该工作台内
- **双写**（既通用又专用）→ 同时写到地平面和工作台

## 近原则

- 不同类型结构相同，只是标签不同 → 统一放在 通用/ 下
- 实际使用时默认用"工作室"，需要区分时再新建类型
- 工作台内专用内容放在工作台内，不污染根目录通用文件夹
