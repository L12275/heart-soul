通用工作台

> 生产空间默认只有一个类型：通用。
> 模型按需自行创建其他类型（工作室、工厂、公司、作坊、创意中心等）。

## 工作台列表

| 工作台 | 用途 |
|--------|------|
| 音乐工作台 | 音频制作、编曲、混音 |
| 视频工作台 | 视频剪辑、后期、特效 |
| 文章工作台 | 写作、文档、报告 |
| 研究工作台 | 深度研究、分析 |
| 代码工作台 | 编程、调试、测试 |
| 项目工作台 | 项目管理、任务跟踪 |
| 科学工作台 | 科学计算、仿真、可视化 |

## 如何创建新类型

需要新类型时，直接在 production_space/ 下新建文件夹即可：
```
production_space/
├── README.md
└── 通用/
    ├── README.md
    ├── 音乐工作台/
    ├── ...
    └── [新类型]/ ← 模型自行创建
        └── [工作台]/
```

## 近原则

- 通用工作台内容 = 所有场景都可能用到
- 专用类型内容 = 只在特定场景使用
- 不需要提前建所有类型，用到时再建
