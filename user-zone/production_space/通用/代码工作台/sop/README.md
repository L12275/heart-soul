# 代码工作台 — SOP

> 标准操作流程。
> 参考GA的SOP自动沉淀与优化系统：https://github.com/lsdefine/GenericAgent

## 编码SOP

### 写代码前
1. 读现有代码逻辑
2. 确认需求和边界
3. 规划实现方案
4. 写测试用例

### 编码中
1. 小步提交，每步可运行
2. 注释关键逻辑
3. 遇到问题记录到 experience/

### 编码后
1. 运行测试
2. 代码审查
3. 更新文档
4. 记录经验到 experience/

## SOP自动优化

- 每次执行SOP后记录实际耗时和问题
- 积累5条验证规则触发SOP更新
- 优化后的SOP标记为 v2、v3...
