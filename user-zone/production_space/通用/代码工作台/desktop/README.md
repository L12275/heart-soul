# 代码工作台 — 桌面

> 编码头脑风暴空间 —— 代码实验、临时调试、快速原型。

**与生产空间的关系：**
- 生产空间 = 正式产出
- 工作台桌面 = 代码实验和头脑风暴
- 验证通过的代码 → 移入 product/执行类/

---

## 目录

| 子目录 | 说明 |
|--------|------|
| sticky-notes/ | 代码待办、调试笔记、临时想法 |
| workflows/ | 编码工作流模板 |
| bookmarks/ | 编码相关收藏夹 |
| contacts/ | 开发工具通讯录 |

## 使用规则

- 临时脚本 → 验证后移入 product/执行类/scripts/
- 调试笔记 → 问题解决后归档到 wiki/
- 工作流模板 → 实际使用后优化并存档
