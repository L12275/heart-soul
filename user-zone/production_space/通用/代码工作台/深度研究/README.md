> 每个研究 = 一个文件夹，命名格式：`[时间戳]_[研究主题]`

## 研究结构

```
深度研究/
├── 2026-06-23_GA记忆系统研究/
│   ├── README.md        ← 研究目标、背景
│   ├── 发现.md          ← 研究发现
│   ├── 结论.md          ← 研究结论
│   └── 参考资料/        ← 相关链接和文档
├── [下一个研究]/
```
