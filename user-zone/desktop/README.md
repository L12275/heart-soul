# 大脑桌面（Desktop）

> 头脑风暴空间 —— 宽裕松散，不是密集干活。
>
> **与生产空间/工作台 的关系：**
> - 生产空间是最大的单位
> - 工作台是密集专注
> - 大脑桌面是宽裕松散
>
> 桌面 = 准备、散布、便签、机会收集

---

## 目录

| 子目录 | 说明 |
|--------|------|
| sticky-notes/ | 零碎提醒、临时便签 |
| workflows/ | 通用工作流模板 |
| contacts/ | 通讯录 |
| bookmarks/ | 收藏夹 |

## 使用说明

- 头脑风暴、实验、编程、数据存放都在此。
- 工作台上已完成的工作，数据可整理到桌面存档。
- 形成从生产空间到桌面的垂直深度。
