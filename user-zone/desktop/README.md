大脑桌面（Desktop）

> 头脑风暴空间 —— 宽裕松散，不是密集干活。
>
> **与生产空间/工作台 的关系：**
> - 生产空间是最大的单位
> - 工作台是密集专注
> - 大脑桌面是宽裕松散
>
> 桌面 = 准备、散布、便签、机会收集
