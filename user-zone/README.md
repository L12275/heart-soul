# 用户专用文件夹（User Zone）

> 此文件夹由用户管理，用于存放用户认为有价值的内容。
> 所有模型都可以读取此文件夹进行学习和参考。
> **只有用户可以增删此文件夹的内容。**

## 目录结构

```
user-zone/
├── README.md          ← 本文件
├── wiki/              ← 知识wiki
├── sop/               ← 用户收藏的SOP
├── 经验/              ← 经验总结
├── 文档/              ← 用户收集的文档
└── 杂收集/            ← 杂项收集
```

## 使用规则

1. **读取**：所有模型都可以自由读取此文件夹的任何内容
2. **学习**：模型读取后，可以从中提取对自己灵魂文档有用的内容
3. **不写入**：模型**禁止**向此文件夹写入、修改、删除任何内容
4. **不引用**：模型在分析此文件夹内容时，不能将内容直接复制到自己的灵魂文档（只能吸收理解后用自己的话重写）
5. **用户整理**：如果用户要求模型帮助整理此文件夹，模型可以做，但删除操作需要**两次确认**，且不能批量删除

## 目录说明

| 目录 | 用途 |
|------|------|
| wiki/ | 用户收集的知识条目、百科内容 |
| sop/ | 用户认为优秀的SOP和工作流程 |
| 经验/ | 用户的经验总结和教训记录 |
| 文档/ | 用户收藏的各类文档 |
| 杂收集/ | 用户收集的其他有价值的内容 |
